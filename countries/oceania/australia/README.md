# 🇦🇺 澳大利亚（Commonwealth of Australia）

> 全称「澳大利亚联邦」，是世界上唯一占据整个大陆的国家，也是大洋洲面积最大、经济最发达的国家。资源极其丰富，社会高度发达、多元，是 G20 成员和重要的矿产、农业出口国，以独特的自然生态和宜居环境闻名。

## 🧭 一句话画像

澳大利亚是地球上唯一独占一块大陆的国家，拥有独特而古老的自然生态（袋鼠、考拉、大堡礁）。它资源极为丰富，是铁矿石、煤炭、天然气和农产品的世界级出口国。作为高度发达的英语国家和移民社会，澳大利亚生活质量位居世界前列，地缘上立足亚太，外交上紧随美英，是 AUKUS 和 QUAD 的重要成员。

## 🗺️ 速读要点

- **大陆国家**：世界唯一独占整块大陆的国家，世界第六大国土面积。
- **资源**：铁矿石、煤炭、天然气、锂、黄金等矿产世界级储量与出口。
- **生态**：长期与世隔绝形成独特物种（有袋类动物），拥有大堡礁等自然奇观。
- **人口**：约 2,700 万，高度集中于东南沿海城市，移民比例极高。
- **政治**：君主立宪下的联邦议会制（威斯敏斯特体系）。
- **国际**：美澳同盟、AUKUS、QUAD 的核心成员，地缘立足亚太。
