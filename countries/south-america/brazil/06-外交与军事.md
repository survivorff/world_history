# 巴西 · 外交与军事

## 1. 外交定位

巴西是**地区大国**，外交以自主、多边和「全球南方」代表自居：

- 奉行独立自主、多边主义的外交传统。
- 自我定位为**拉美与「全球南方」的领导者与代言人**。
- 积极参与全球治理，长期争取成为联合国安理会常任理事国。
- 重视南南合作与发展中国家团结。
- 主张通过和平方式解决国际争端，不结盟。

---

## 2. 主要双边与地区关系

| 对象 | 关系概述 |
| --- | --- |
| **中国** | 最大贸易伙伴，农产品与矿产的最大买家，金砖伙伴 |
| **美国** | 重要的贸易与政治伙伴 |
| **阿根廷** | 南美核心邻国，南共市重要伙伴 |
| **南美邻国** | 通过南方共同市场（Mercosur）等加强一体化 |
| **非洲、葡语国家** | 重视南南合作与葡语国家联系 |

---

## 3. 国际组织与多边参与

- **联合国**：创始会员国，长期争取安理会常任理事国席位（与德、日、印组成「四国集团」G4）。
- **金砖国家（BRICS）**：创始成员，「B」即代表巴西。
- **G20**：重要成员。
- **南方共同市场（Mercosur）**：核心成员，南美一体化机制。
- **拉共体（CELAC）、美洲国家组织（OAS）**等区域组织。

---

## 4. 军事力量

巴西武装力量是拉丁美洲规模最大的军队。

| 项目 | 概况 |
| --- | --- |
| 规模 | 拉美最大的军队 |
| 军种 | 陆军、海军、空军 |
| 核武器 | **无核武器**，签署核不扩散条约，宪法承诺核能仅用于和平目的 |
| 国防工业 | 有一定基础（巴西航空工业公司亦涉军机） |
| 特点 | 主要承担国土防卫、亚马孙地区管控与维和任务 |

- 巴西积极参与联合国维和行动（曾长期领导联合国海地维和部队）。
- 正发展国产潜艇等装备。

---

## 5. 安全与外交焦点

- **亚马孙保护**已成为巴西外交的重要议题，与气候外交紧密相连。
- 在金砖、G20 等平台推动「全球南方」诉求与国际秩序改革。
- 维护南美地区稳定与一体化。
- 平衡与中国、美国的大国关系。

---

## 6. 软实力与对外影响

- **足球**作为世界级文化名片。
- 桑巴、狂欢节、波萨诺瓦音乐的全球影响。
- 葡语国家的文化纽带。
- 在环境与气候议题上的全球分量（亚马孙）。

---

## 📌 外交与军事小结

巴西以地区大国和「全球南方」领导者自居，奉行自主、多边的外交路线，是金砖国家和南方共同市场的核心成员，并长期谋求联合国安理会常任理事国地位。其军队规模为拉美之最但不拥核，以国土防卫和维和为主。足球与桑巴构成的软实力，以及亚马孙带来的全球环境分量，是巴西独特的国际影响力来源。
