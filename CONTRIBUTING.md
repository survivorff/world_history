# 贡献指南

感谢你愿意为「世界国家与地区知识库」贡献内容！本指南帮助你快速上手。

## 🎯 贡献原则

1. **客观中立**：以描述性、事实性的语言陈述，避免主观评判或情绪化表达，尤其在历史与政治议题上。
2. **来源可靠**：数据优先采用官方统计、国际组织（联合国、世界银行、IMF 等）、权威百科与学术来源。
3. **标注时效**：所有统计数据需标注**参考年份**（如「GDP：约 27 万亿美元（2023 年）」）。
4. **结构统一**：新增国家请遵循 [国家介绍模板](templates/国家介绍模板.md)。

## 📂 目录与命名规范

- 国家目录路径：`countries/<大洲英文>/<国家英文小写连字符>/`
  - 大洲：`north-america`、`south-america`、`europe`、`asia`、`africa`、`oceania`
  - 例：`countries/asia/japan/`
- 专题文件采用编号前缀，保证排序一致：
  ```
  README.md          速览页
  01-地理.md
  02-历史.md
  03-政治.md
  04-经济.md
  05-文化与社会.md
  06-外交与军事.md
  ```

## ✍️ 写作规范

- 使用简体中文为主，专有名词可附英文原文，如「国会（Congress）」。
- 标题层级清晰，多用表格、列表整理数据，便于阅读。
- 每个文档顶部提供「返回国家目录 / 返回主索引」导航链接。
- 数字尽量给出单位与年份；引用观点时注明来源。

## 🔄 提交流程

1. Fork 本仓库并创建分支：`git checkout -b add-country-japan`
2. 按模板补充内容。
3. 提交 PR，标题简洁说明改动（如 `新增国家：日本`）。
4. 在 PR 描述中列出主要数据来源。

## ✅ 内容检查清单

提交前请自查：

- [ ] 是否遵循统一目录结构与文件命名？
- [ ] 关键数据是否标注年份与来源？
- [ ] 表述是否客观中立？
- [ ] 是否更新了主 `README.md` 的国家索引表？
- [ ] 链接是否可正常跳转？
