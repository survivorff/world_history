## 1. 地理位置与疆域

巴西位于**南美洲东中部**，是南美洲最大的国家，占据该大陆近一半的面积。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 851 万平方公里（世界第 5，南美第 1） |
| 陆地邻国 | 除厄瓜多尔和智利外，与南美所有国家接壤（10 个） |
| 濒临海域 | 大西洋 |
| 海岸线 | 约 7,491 公里 |

- 邻国包括：阿根廷、乌拉圭、巴拉圭、玻利维亚、秘鲁、哥伦比亚、委内瑞拉、圭亚那、苏里南、法属圭亚那。
- 东临大西洋，拥有漫长的海岸线。
- 赤道与南回归线均穿过巴西，国土大部分位于热带。

## 2. 地形地貌

巴西地形主要由**高原和平原**构成：

| 区域 | 特征 |
| --- | --- |
| **亚马孙平原** | 北部，世界最大的平原与热带雨林 |
| **巴西高原** | 中南部，占国土大部，海拔较低的古老高原 |
| 沿海地带 | 大西洋沿岸狭长平原 |

- 地形总体起伏不大，无高大山系。
- 最高峰：内布利纳峰（约 2,995 米），位于北部边境。

### 主要河流
- **亚马孙河（Amazon）**：世界流量最大、流域最广的河流，干流大部分在巴西境内。
- 巴拉那河、圣弗朗西斯科河等。

---

## 3. 亚马孙雨林（地球之肺）

- **亚马孙雨林**是世界最大的热带雨林，约 60% 位于巴西境内。
- 拥有地球上最丰富的**生物多样性**，被称为「地球之肺」。
- 对全球气候调节、碳储存和水循环至关重要。
- 面临**森林砍伐、火灾和开发**的威胁，其保护是国际环境议题的焦点。

---

## 4. 气候

巴西大部分位于热带，气候多样：

| 气候类型 | 分布 |
| --- | --- |
| 热带雨林气候 | 亚马孙地区，全年高温多雨 |
| 热带草原气候 | 中部高原，分干湿两季 |
| 半干旱气候 | 东北部「干旱多边形」地区 |
| 亚热带气候 | 南部，四季较分明，偶有霜冻 |

## 5. 自然资源

巴西自然资源极为丰富：

- **矿产**：铁矿石（储量与产量世界前列）、铝土、锰、黄金、宝石。
- **能源**：石油（深海盐下油田）、丰富的水力资源（伊泰普水电站）、生物燃料（甘蔗乙醇）。
- **农业用地**：广袤的可耕地与牧场，是农业大国的基础。
- **森林与生物**：亚马孙的生物资源不可估量。
- **淡水**：拥有世界上最丰富的淡水资源之一。

## 6. 行政区划

巴西是**联邦制**国家，由 **26 个州 + 1 个联邦区**组成：

- 联邦区为首都巴西利亚所在地。
- 各州拥有自己的州长、州议会与较大自治权。
- 下设市镇（Municípios）。

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 巴西利亚（Brasília） | 首都，1960 年建成的规划城市 |
| 圣保罗（São Paulo） | 最大城市，南半球最大都市之一，经济中心 |
| 里约热内卢（Rio de Janeiro） | 著名旅游城市，文化与海滨名城 |
| 萨尔瓦多（Salvador） | 历史名城，非洲裔文化中心 |
| 贝洛奥里藏特、福塔莱萨 | 重要区域中心 |

## 7. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 2.16 亿（2024 年，世界第 7） |
| 城市化率 | 约 88% |
| 分布 | 高度集中于东南沿海地区 |

- 人口高度集中在**东南部和东北沿海**，尤其圣保罗与里约都市圈。
- 内陆（尤其亚马孙地区）人口稀疏。
- 大城市存在大规模的贫民窟（favela）。

---

## 📌 小结

巴西国土辽阔，地处热带，坐拥世界最大的亚马孙雨林和亚马孙河，自然资源得天独厚——铁矿、农地、水力、石油样样丰富。人口与经济高度集中于东南沿海，而广袤的内陆与雨林则人烟稀少。亚马孙雨林的保护与开发，是巴西面对的国家与全球双重课题。
