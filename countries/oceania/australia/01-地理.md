## 1. 地理位置与疆域

澳大利亚位于**南半球**，是大洋洲主体，也是世界上**唯一独占一整块大陆**的国家。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 769 万平方公里（世界第 6） |
| 陆地邻国 | 无（四面环海） |
| 濒临海域 | 印度洋、太平洋、南大洋 |
| 海岸线 | 约 3.4 万公里 |
| 主要岛屿 | 塔斯马尼亚岛（南部） |

- 四面环海，与亚洲（印尼、东帝汶等）隔海相望。
- 地理上长期相对孤立，造就了独特的生态系统。

---

## 2. 地形地貌

澳大利亚是地球上**最平坦、最干燥**的有人居住大陆之一：

| 区域 | 特征 |
| --- | --- |
| 东部 | 大分水岭（Great Dividing Range）山系，沿海较湿润 |
| 中部 | 广袤干旱的内陆「内陆腹地（Outback）」与沙漠 |
| 西部 | 古老的高原与沙漠 |
| 沿海 | 平原与人口聚居带 |

- 最高峰：科修斯科山（Mount Kosciuszko，约 2,228 米），相对较低。
- 著名地标：**乌鲁鲁（艾尔斯岩 Uluru）**，巨大的红色砂岩巨石。
- **大堡礁（Great Barrier Reef）**：东北沿海，世界最大的珊瑚礁群，自然奇观。

## 3. 气候

气候多样，但总体**干旱**：

| 区域 | 气候 |
| --- | --- |
| 北部 | 热带气候（分干湿季） |
| 中部内陆 | 沙漠与半干旱气候，极度干燥 |
| 东南/西南沿海 | 温带气候，较适宜居住 |
| 塔斯马尼亚 | 温带海洋性气候 |

- 大部分国土为沙漠或半干旱地区。
- 易受干旱、丛林大火（Bushfire）和洪水等灾害影响。
- 南半球季节与北半球相反。

## 4. 生态与自然资源

### 独特生态
- 长期地理隔离形成独有物种：**袋鼠、考拉、鸭嘴兽、袋熊**等有袋类与单孔类动物。
- 桉树等特有植物广布。
- 生物多样性独特但生态脆弱。

### 自然资源（极其丰富）
- **矿产**：铁矿石（世界最大出口国）、煤炭、铝土（世界最大）、黄金、铀、锂、镍。
- **能源**：液化天然气（世界主要出口国）、煤炭。
- **农牧业**：广阔牧场（羊毛、牛肉世界领先）、小麦。

## 5. 行政区划

澳大利亚是**联邦制**国家，由 **6 个州 + 2 个领地**组成：

| 类型 | 名称 |
| --- | --- |
| 6 个州 | 新南威尔士、维多利亚、昆士兰、西澳大利亚、南澳大利亚、塔斯马尼亚 |
| 2 个领地 | 北领地、澳大利亚首都领地（堪培拉所在） |

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 堪培拉（Canberra） | 首都，规划建成的政治中心 |
| 悉尼（Sydney） | 最大城市，金融与商业中心 |
| 墨尔本（Melbourne） | 第二大城市，文化与体育中心 |
| 布里斯班（Brisbane） | 昆士兰首府 |
| 珀斯（Perth） | 西澳门户，矿业中心 |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 2,700 万（2024 年） |
| 人口密度 | 极低（约 3.5 人/平方公里） |
| 城市化率 | 约 86% |

- 人口高度集中于**东南沿海**（悉尼、墨尔本、布里斯班）和西南角（珀斯）。
- 广袤的内陆几乎无人居住。
- 是移民比例极高的国家，约近三成人口在海外出生。

---

## 📌 小结

澳大利亚独占一块古老、平坦而干旱的大陆，中部是荒凉的「内陆腹地」，人口与城市紧贴东南沿海。长期的地理隔离孕育了袋鼠、考拉等独一无二的生态，而铁矿、煤炭、天然气等惊人的资源禀赋，则是其繁荣的物质基础。大堡礁和乌鲁鲁是其享誉世界的自然名片。
