# 🇩🇪 德国（Federal Republic of Germany）

> 全称「德意志联邦共和国」，位于欧洲中部，是欧盟人口最多、经济体量最大的国家，世界第三大经济体（名义 GDP）。以高端制造、工程技术、出口导向和「社会市场经济」著称，是欧洲一体化的核心引擎。

## 🧭 一句话画像

德国是欧洲的经济与工业中心，以汽车、机械、化工等高端制造和强大的出口能力闻名于世。它在经历了 20 世纪两次世界大战的创伤与分裂后，于 1990 年重新统一，如今是欧盟的领导力量、稳定的议会制民主国家，并以「德国制造」品质和严谨务实的国民性格享誉全球。

## 🗺️ 速读要点

- **经济**：欧盟最大经济体、世界第三大经济体，出口导向型，以制造业为根基。
- **政治**：联邦议会制共和国，以《基本法》为宪法，总理为政府核心。
- **历史**：1871 年首次统一，经历两次世界大战，1949 年分裂为东西德，1990 年重新统一。
- **制造业**：汽车（大众、宝马、奔驰）、机械、化工、隐形冠军「中小企业（Mittelstand）」。
- **文化**：哲学（康德、黑格尔、马克思）、音乐（巴赫、贝多芬）、汽车与工程文化。
- **欧盟**：欧洲一体化的核心驱动力之一，与法国构成「欧盟轴心」。
