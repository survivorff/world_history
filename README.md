# 世界国家与地区知识库 🌍

> 一个系统化、结构化的全球各国家与地区**历史与现状**知识库。从影响力最大的国家开始，全方位梳理每个国家的**地理、历史、政治、经济、文化、外交与军事**等信息，力求详尽、客观、可持续更新。

## 📖 项目简介

本仓库旨在打造一个**人人可读、人人可贡献**的全球国情知识库。每个国家都按统一模板拆分为多个专题文档，既方便快速查阅，也便于深入研究。

### 内容覆盖维度

每个国家/地区包含以下专题：

| 专题 | 内容概要 |
| --- | --- |
| 📍 地理 | 位置、地形、气候、自然资源、行政区划、人口分布 |
| 📜 历史 | 从古代到现代的关键历史脉络与重大事件 |
| 🏛️ 政治 | 政治体制、政府结构、政党、法律、选举制度 |
| 💰 经济 | 经济体量、产业结构、货币、贸易、发展水平 |
| 🎭 文化与社会 | 语言、宗教、民族、教育、艺术、风俗、社会议题 |
| 🛡️ 外交与军事 | 国际关系、军事力量、重要条约与组织成员身份 |

## 🧭 如何使用

- 点击上方索引表中的链接进入对应国家目录。
- 每个国家目录下的 `README.md` 是**速览页**，包含核心数据与各专题导航。
- 想深入了解某一方面，直接进入编号文档（如 `02-历史.md`）。
