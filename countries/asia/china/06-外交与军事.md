## 1. 外交方针

中国奉行**独立自主的和平外交政策**，核心理念包括：

- **和平共处五项原则**：互相尊重主权和领土完整、互不侵犯、互不干涉内政、平等互利、和平共处。
- 坚持走**和平发展道路**。
- 推动构建**人类命运共同体**。
- 维护以联合国为核心的国际体系和以国际法为基础的国际秩序。
- 坚持**不结盟**，但发展全球伙伴关系网络。

## 2. 主要双边与地区关系

| 对象 | 关系概述 |
| --- | --- |
| **美国** | 最重要也最复杂的双边关系，竞争与合作并存，涉及贸易、科技、台湾、地缘安全 |
| **俄罗斯** | 「新时代全面战略协作伙伴关系」，能源、安全合作密切 |
| **欧盟** | 重要经贸伙伴，合作与分歧并存 |
| **周边国家** | 倡导「亲诚惠容」，与东盟经贸联系紧密；与部分邻国存在领土/海洋争议 |
| **发展中国家** | 重视与「全球南方」合作，金砖、上合等机制 |
| **日本、韩国** | 重要经贸伙伴，历史与安全问题时有摩擦 |

### 核心利益问题
- **台湾问题**被视为核心利益中的核心，坚持一个中国原则。
- 在南海、钓鱼岛等存在主权与海洋权益主张。

## 3. 国际组织与多边参与

- **联合国**：创始会员国，**安理会五常之一**（拥有否决权），第二大会费与维和摊款来源国，派出大量维和人员。
- **G20**：主要成员。
- **金砖国家（BRICS）**：核心成员，推动扩员与「全球南方」合作。
- **上海合作组织（SCO）**：创始成员，地区安全与合作机制。
- **WTO、IMF、世界银行、APEC**等。
- 发起设立**亚洲基础设施投资银行（AIIB）**与「一带一路」合作机制。

## 4. 军事力量

中国人民解放军（PLA）是世界规模最大的军队之一，由中国共产党绝对领导（中央军事委员会统一指挥）。

| 项目 | 概况 |
| --- | --- |
| 军费 | 世界第二（约占世界第二位） |
| 现役兵力 | 约 200 万人，世界规模最大之一 |
| 军种 | 陆军、海军、空军、火箭军，及战略支援、联勤保障等 |
| 核力量 | 拥有核武器，奉行「不首先使用核武器」政策 |
| 海军 | 舰艇数量快速增长，已成为世界规模最大的海军之一，拥有多艘航空母舰 |

- 近年来推进**国防和军队现代化**，目标是建成「世界一流军队」。
- 在航天、导弹、隐身战机（歼-20）、航母、无人系统等领域快速发展。
- 设立海外保障基地（如吉布提）。

## 5. 重要倡议与安全理念

- **全球发展倡议（GDI）**、**全球安全倡议（GSI）**、**全球文明倡议（GCI）**。
- **一带一路**：以基础设施互联互通促进沿线国家共同发展。
- 强调「**共同、综合、合作、可持续的安全观**」。

## 6. 软实力与对外影响

- 通过孔子学院、文化交流、影视、短视频平台（如 TikTok）扩大文化影响。
- 在基建、新能源、数字经济等领域对发展中国家影响上升。
- 大量留学生与海外华人构成文化纽带。

## 📌 外交与军事小结

中国以独立自主的和平外交为基调，在维护核心利益的同时积极参与全球治理，推动构建人类命运共同体。随着综合国力上升，中国军事现代化加速、国际话语权增强，与美国等大国的战略互动正深刻影响 21 世纪的世界格局。
