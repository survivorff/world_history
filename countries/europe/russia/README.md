# 🇷🇺 俄罗斯（Russian Federation）

> 全称「俄罗斯联邦」，是世界上**陆地面积最大的国家**，横跨欧亚两大洲、11 个时区。拥有丰富的能源与矿产资源，是联合国安理会常任理事国和核大国，在国际政治与能源市场中具有重要影响力。

## 🧭 一句话画像

俄罗斯是世界上幅员最辽阔的国家，继承了苏联的大部分遗产，拥有庞大的核武库和丰富的石油、天然气资源。它横跨欧亚，既是欧洲国家又是亚洲大国，在能源、军事和国际安全事务中举足轻重，近年来因乌克兰危机与西方关系高度紧张。

## 🗺️ 速读要点

- **国土**：世界第一大国，面积约占全球陆地的 1/8，横跨 11 个时区。
- **资源**：天然气储量世界第一，石油、煤炭、矿产、森林资源极其丰富。
- **政治**：半总统制，普京长期主导政坛。
- **军事**：拥有世界最大核武库之一，军力位居世界前列。
- **历史**：继承沙俄与苏联遗产，1991 年苏联解体后成立俄罗斯联邦。
- **当代**：2022 年起的俄乌冲突使其与西方关系陷入冷战后最低点，遭受大规模制裁。
