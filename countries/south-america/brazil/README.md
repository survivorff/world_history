# 🇧🇷 巴西（Federative Republic of Brazil）

> 全称「巴西联邦共和国」，是南美洲面积最大、人口最多的国家，世界第五大国土面积国。拥有亚马孙雨林、丰富的自然资源和多元的种族文化，是拉丁美洲最大经济体、金砖国家成员和重要的农业与矿业大国。

## 🧭 一句话画像

巴西是南美洲的「巨人」，国土与人口均居南美之首，是拉美最大的经济体。它坐拥地球之肺亚马孙雨林和丰富的矿产、农业资源，是世界主要的大豆、咖啡、铁矿石和牛肉出口国。多元种族融合的文化孕育了桑巴、狂欢节和足球王国的盛名，是金砖国家和「全球南方」的重要代表。

## 🗺️ 速读要点

- **国土**：世界第五大国，占南美洲近一半面积，拥有全球最大的亚马孙雨林。
- **经济**：拉美第一大经济体，农业（大豆、咖啡、牛肉）与矿业（铁矿）出口大国。
- **政治**：联邦总统制共和国，民主制度在 1985 年结束军政府后重建。
- **人口**：种族高度融合，是世界上葡萄牙语人口最多的国家。
- **文化**：足球王国（5 次世界杯冠军）、桑巴、里约狂欢节享誉全球。
- **环境**：亚马孙雨林的保护与开发是全球关注的焦点。
