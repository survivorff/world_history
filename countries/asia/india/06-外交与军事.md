## 1. 外交定位

印度外交以**战略自主（Strategic Autonomy）**为核心理念，延续独立后的**不结盟传统**：

- 是**不结盟运动（NAM）**的创始与领导国之一。
- 奉行**多向结盟/多边平衡**外交，不加入任何军事集团，与各大国均保持往来。
- 自我定位为**「全球南方」（Global South）**的代表与领导者。
- 追求成为**有声有色的世界大国**与联合国安理会常任理事国。

---

## 2. 主要双边与地区关系

| 对象 | 关系概述 |
| --- | --- |
| **美国** | 关系日益紧密，在防务、科技、印太战略上深化合作 |
| **俄罗斯** | 传统友好，重要的武器与能源来源国 |
| **中国** | 既是重要贸易伙伴，又是边境争端与地缘竞争对手 |
| **巴基斯坦** | 长期对手，围绕克什米尔问题多次冲突，关系紧张 |
| **邻国（孟、尼、斯里兰卡等）** | 推行「邻国优先」政策，影响力与竞争并存 |
| **欧盟、日本、澳大利亚** | 不断加强的合作伙伴 |

---

## 3. 国际组织与多边参与

- **联合国**：重要会员国，积极争取成为安理会常任理事国，是主要的维和部队贡献国之一。
- **G20**：重要成员，2023 年担任轮值主席国。
- **金砖国家（BRICS）**：创始成员。
- **上海合作组织（SCO）**：成员国。
- **QUAD（四方安全对话）**：美日印澳协调机制成员。
- **英联邦**成员。

---

## 4. 军事力量

印度武装力量规模位居世界前列。

| 项目 | 概况 |
| --- | --- |
| 兵力 | 现役兵力世界最大之一（约 140 万） |
| 核力量 | **核大国**（1974、1998 年核试验），奉行「不首先使用」政策 |
| 军种 | 陆军、海军、空军 |
| 军费 | 居世界前列 |
| 武器来源 | 长期依赖俄罗斯，近年来源多元化并推动国产化 |

- 拥有航空母舰、弹道导弹（烈火系列）、核潜艇等。
- 推动「**国防自主（自力更生）**」，发展本土国防工业。

## 5. 安全与外交焦点

- **中印边境**：存在领土争议，曾发生对峙与冲突。
- **印巴关系**：克什米尔问题与反恐议题。
- 通过 QUAD 等强化在印太地区的作用。
- 在大国之间保持平衡：既深化与美国合作，又维系与俄罗斯的传统关系。
- 海洋安全与印度洋地区影响力。

## 6. 软实力与对外影响

- **瑜伽**（国际瑜伽日）、阿育吠陀等传统文化的全球传播。
- 宝莱坞电影在南亚、中东、非洲的广泛影响。
- 庞大的海外印度裔群体（Diaspora）构成全球纽带。
- IT 与人才输出的国际声誉。

## 📌 外交与军事小结

印度以「战略自主」和不结盟传统为外交基石，在美、俄、中等大国之间灵活平衡，并以「全球南方」领导者自居。作为核大国和军事规模名列世界前茅的国家，印度正通过 QUAD、金砖、G20 等多边平台扩大影响力，谋求与其人口和经济体量相称的世界大国地位。
