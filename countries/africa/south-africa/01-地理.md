## 1. 地理位置与疆域

南非位于**非洲大陆最南端**，扼守大西洋与印度洋之间的航道。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 122 万平方公里（世界第 24） |
| 陆地邻国 | 纳米比亚、博茨瓦纳、津巴布韦、莫桑比克、斯威士兰（埃斯瓦蒂尼） |
| 国中之国 | **莱索托**（被南非完全包围） |
| 濒临海域 | 大西洋（西）、印度洋（东南） |

- 位于非洲最南端，扼守**好望角**航线，连接大西洋与印度洋，战略位置重要。

## 2. 地形地貌

南非地形以**内陆高原**为主：

| 区域 | 特征 |
| --- | --- |
| 内陆 | 广阔的高原（Highveld），平均海拔较高 |
| 边缘 | **大悬崖（Great Escarpment）**，高原向沿海陡降 |
| 东部 | 德拉肯斯山脉（Drakensberg），最高山系 |
| 沿海 | 狭窄的沿海平原 |

- 最高点：德拉肯斯山脉的高峰（超过 3,400 米）。
- 西部有卡拉哈里沙漠边缘和干旱地区。
- 主要河流：奥兰治河（最长）、林波波河。

## 3. 气候

南非气候多样，总体温和：

| 区域 | 气候 |
| --- | --- |
| 西南（开普敦） | 地中海气候，夏干冬雨 |
| 东部沿海 | 亚热带，温暖湿润 |
| 内陆高原 | 温带、半干旱，日照充足 |
| 西部 | 干旱、沙漠 |

- 位于南半球，季节与北半球相反。
- 水资源相对短缺，部分地区易受干旱影响。

## 4. 自然资源（矿产宝库）

南非是世界级的**矿产资源大国**：

- **铂族金属（PGM）**：储量与产量**世界第一**。
- **黄金**：曾是世界最大产金国，储量仍极丰富。
- **钻石**：著名的钻石产地（戴比尔斯公司发源地）。
- **锰、铬**：储量世界领先。
- **煤炭**：丰富，是主要能源与出口品。
- 矿业是南非工业化与经济的历史基石。

## 5. 行政区划

南非由 **9 个省**组成：

- 豪登省（约翰内斯堡、比勒陀利亚所在，经济核心）、西开普省、夸祖鲁-纳塔尔省、东开普省等。
- 实行独特的**三个首都**制度（分权设计）。

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 约翰内斯堡（Johannesburg） | 最大城市，经济与金融中心 |
| 开普敦（Cape Town） | 立法首都，旅游名城 |
| 比勒陀利亚（Pretoria） | 行政首都 |
| 德班（Durban） | 重要港口城市 |
| 布隆方丹（Bloemfontein） | 司法首都 |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 6,300 万（2024 年） |
| 城市化率 | 约 68% |
| 构成 | 黑人约 81%，有色人种、白人、亚裔等 |

- 人口集中于**豪登省**（约翰内斯堡—比勒陀利亚都市圈）和沿海大城市。
- 黑人是人口主体，社会种族构成多元。
- 城乡与种族间的居住与经济差距仍较明显（种族隔离遗留影响）。

## 📌 小结

南非雄踞非洲大陆最南端，扼守好望角连接两洋的航线。内陆高原构成国土主体，气候温和多样。这里是世界级的矿产宝库——铂金、黄金、钻石、煤炭储量惊人，矿业奠定了南非工业化的基础。独特的「三个首都」设计和多元的人口构成，体现了这个「彩虹之国」的复杂性。
