# 🇫🇷 法国（French Republic）

> 全称「法兰西共和国」，位于西欧，是欧盟核心成员、联合国安理会常任理事国与核大国。以悠久的文化艺术、启蒙思想、时尚奢侈品、美食和强大的航空航天工业著称，是世界主要发达经济体之一。

## 🧭 一句话画像

法国是欧洲的文化与政治大国，启蒙运动和大革命的发源地，「自由、平等、博爱」的理念深刻影响了现代世界的民主与人权观念。它是联合国安理会常任理事国和核大国，拥有强大的航空航天、核能、奢侈品和农业产业，并以艺术、美食、时尚和「法式生活方式」享誉全球，是世界第一大旅游目的地。

## 🗺️ 速读要点

- **政治**：半总统制，总统权力较大，现行第五共和国宪法（1958 年）。
- **历史**：启蒙运动与 1789 年大革命的发源地，深刻影响现代政治理念。
- **经济**：世界主要经济体，航空（空客）、核能、奢侈品、农业与旅游强国。
- **军事**：核大国，安理会常任理事国，欧盟防务的核心力量。
- **文化**：艺术、文学、哲学、时尚、美食的世界中心之一，世界第一旅游大国。
- **欧盟**：与德国并称欧盟一体化的「双引擎」。
