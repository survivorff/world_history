## 1. 外交定位

意大利外交以**欧洲一体化**与**跨大西洋同盟**为两大支柱：

- 是**欧盟创始国**之一（1957 年《罗马条约》在罗马签署）。
- 坚定的**北约**成员，与美国保持紧密同盟。
- 高度重视**地中海**地区事务（移民、能源、安全）。
- 积极参与多边主义与联合国事务。

---

## 2. 主要双边与地区关系

| 对象 | 关系概述 |
| --- | --- |
| **欧盟伙伴** | 与德、法同为欧盟核心，深度融入一体化 |
| **美国** | 重要盟友，北约框架下安全合作紧密 |
| **地中海/北非国家** | 因移民、能源（天然气）问题关系重要 |
| **梵蒂冈** | 国中之国，关系特殊（《拉特兰条约》） |
| **中国、俄罗斯** | 经贸往来与立场协调中保持自主 |

---

## 3. 国际组织与多边参与

- **欧盟（EU）**：创始成员国。
- **联合国**：重要会员国与维和贡献国。
- **G7 / G20**：核心成员。
- **NATO（北约）**：创始成员，南欧重要支柱，驻有美军基地。
- **WTO、OECD** 等。

---

## 4. 军事力量

意大利武装力量是欧洲和北约的重要组成部分。

| 项目 | 概况 |
| --- | --- |
| 军种 | 陆军、海军、空军、卡宾枪骑兵（宪兵 Carabinieri） |
| 核武器 | **自身无核武器**，参与北约「核共享」 |
| 海军 | 拥有航空母舰，地中海地区重要海上力量 |
| 国防工业 | 较强（莱昂纳多 Leonardo 是欧洲主要防务企业之一） |
| 任务 | 北约义务、地中海安全、国际维和 |

- 积极参与联合国与北约的国际维和、维稳行动。
- 在地中海的海上安全与反偷渡行动中作用突出。

---

## 5. 安全与外交焦点

- **地中海移民问题**：是难民/移民跨海抵达欧洲的主要登陆国之一。
- 北非（利比亚等）局势与能源供应安全。
- 欧盟一体化与财政、移民政策协调。
- 支持北约对乌克兰的立场。

---

## 6. 软实力与对外影响

- 无与伦比的文化遗产与艺术影响力。
- 时尚、设计、美食、汽车的全球品牌。
- 天主教中心（梵蒂冈位于罗马）带来的特殊影响。
- 世界级旅游吸引力。

---

## 📌 外交与军事小结

意大利作为欧盟创始国和北约重要成员，外交立足于欧洲一体化与跨大西洋同盟，并特别关注地中海地区的移民、能源与安全。其军队是北约南翼的重要力量，拥有较强的国防工业和地中海海上力量。深厚的文化遗产与「意大利制造」的全球魅力，是其软实力的核心来源。
