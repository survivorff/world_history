# 🇿🇦 南非（Republic of South Africa）

> 全称「南非共和国」，位于非洲大陆最南端，是非洲最发达、工业化程度最高的经济体之一。以丰富的矿产（黄金、铂金、钻石）、「彩虹之国」的多元社会和废除种族隔离的历史转型著称，是金砖国家和 G20 中唯一的非洲成员。

## 🧭 一句话画像

南非是非洲大陆最发达的经济体之一，坐拥黄金、铂金、钻石等世界级矿产。它以结束种族隔离（Apartheid）、在曼德拉领导下实现和平民主转型而闻名于世，被誉为「彩虹之国」。作为金砖国家和 G20 中唯一的非洲成员，南非是非洲在国际舞台的重要代表，但也面临高失业、贫富悬殊和社会不平等等严峻挑战。

## 🗺️ 速读要点

- **矿产**：黄金、铂金（世界最大）、钻石、锰、铬等世界级储量。
- **历史**：以结束种族隔离、曼德拉领导的和平转型闻名。
- **经济**：非洲最发达经济体之一，工业化程度高，金融业发达。
- **多元**：12 种官方语言，「彩虹之国」的多元种族社会。
- **三个首都**：行政（比勒陀利亚）、立法（开普敦）、司法（布隆方丹）。
- **挑战**：极高的失业率、世界最严重的贫富差距之一、电力短缺。
