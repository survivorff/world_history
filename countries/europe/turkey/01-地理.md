## 1. 地理位置与疆域

土耳其是**地跨欧亚两洲**的国家，主体（安纳托利亚半岛）在亚洲，小部分（色雷斯）在欧洲。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 78.4 万平方公里 |
| 亚洲部分 | 安纳托利亚（小亚细亚），约占 97% |
| 欧洲部分 | 东色雷斯，约占 3% |
| 陆地邻国 | 希腊、保加利亚、格鲁吉亚、亚美尼亚、阿塞拜疆（飞地）、伊朗、伊拉克、叙利亚 |
| 濒临海域 | 黑海、地中海、爱琴海、马尔马拉海 |

- **博斯普鲁斯海峡**和**达达尼尔海峡**连接黑海与地中海，是世界最重要的海上咽喉之一。
- 海峡将欧亚分开，伊斯坦布尔横跨两岸。

---

## 2. 地形地貌

土耳其以**高原和山地**为主：

| 区域 | 特征 |
| --- | --- |
| 中部 | 安纳托利亚高原 |
| 沿海 | 黑海、爱琴海、地中海沿岸平原与丘陵 |
| 东部 | 多山，海拔较高 |

- 最高峰：**阿勒山（Mount Ararat）**，约 5,137 米（传说中诺亚方舟停靠处）。
- 位于地震活跃带，地震频发（2023 年东南部发生重大地震）。
- 卡帕多奇亚等地有独特的火山地貌。

## 3. 气候

气候多样：

| 区域 | 气候 |
| --- | --- |
| 沿海（地中海、爱琴海） | 地中海气候，夏干热冬温和 |
| 黑海沿岸 | 温和湿润，降水多 |
| 内陆高原 | 大陆性气候，冬冷夏热，温差大 |
| 东部山区 | 寒冷，冬季漫长 |

---

## 4. 自然资源

- 矿产：**硼**（储量世界第一）、铬、铜、煤炭、大理石。
- 农业资源丰富，是少数粮食基本自给的国家之一（榛子、无花果、葡萄干产量世界领先）。
- 水力资源较丰富（底格里斯河、幼发拉底河上游，安纳托利亚东南部工程 GAP）。
- 油气资源有限，能源较依赖进口。

---

## 5. 行政区划

土耳其是**单一制**国家，划分为 **81 个省（İl）**：

- 省下设县（İlçe）。
- 各省由中央任命省长，地方设民选市长与议会。

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 安卡拉（Ankara） | 首都，政治中心 |
| 伊斯坦布尔（Istanbul） | 最大城市，经济文化中心，横跨欧亚 |
| 伊兹密尔（Izmir） | 爱琴海重要港口 |
| 布尔萨、安塔利亚 | 工业 / 旅游重镇 |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 8,550 万（2024 年） |
| 城市化率 | 约 77% |
| 人口结构 | 相对年轻，但增速放缓 |

- 人口集中于**伊斯坦布尔（约 1,500 万以上）**、安卡拉、伊兹密尔等大城市与沿海地区。
- 东部相对人口稀疏。
- 收容了大量来自叙利亚等地的难民（数百万）。

---

## 📌 小结

土耳其是名副其实的「欧亚桥梁」，安纳托利亚高原构成国土主体，博斯普鲁斯与达达尼尔海峡扼守黑海通往地中海的咽喉，赋予其无可替代的战略地位。多山而地震频发，资源中以硼矿著称、农业可观。横跨两大洲的伊斯坦布尔，是这一独特地理的最佳象征。
