## 1. 地理位置与疆域

日本位于**亚洲东部、太平洋西北部**，是一个由列岛构成的岛国。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 37.8 万平方公里 |
| 岛屿数量 | 约 1.4 万个（含大小岛屿） |
| 四大岛 | 本州、北海道、九州、四国 |
| 海上邻国 | 隔海与韩国、朝鲜、中国、俄罗斯相望 |
| 濒临海域 | 太平洋、日本海、东海、鄂霍次克海 |

- 没有陆地邻国，四面环海。
- 本州是最大岛屿，集中了主要人口与城市。
- 与周边国家存在领土争议（如与俄罗斯的南千岛群岛/北方四岛、与韩国的独岛/竹岛）。

## 2. 地形地貌

日本地形以**山地丘陵为主**，约占国土的四分之三：

- 平原狭小，主要集中在沿海，如关东平原（东京一带）、浓尾平原、大阪平原。
- 多火山，位于环太平洋火山地震带（「火环」）上。
- 最高峰：**富士山（Mount Fuji）**，海拔约 3,776 米，是日本的象征。
- 河流短促湍急，湖泊有琵琶湖（最大湖泊）。

### 自然灾害
- **地震**频发，是世界上地震最多的国家之一。
- **火山**活动活跃。
- **海啸**风险高（如 2011 年东日本大地震引发的大海啸）。
- 夏秋季多**台风**。

## 3. 气候

日本南北狭长，气候多样，多属**温带季风气候**：

| 区域 | 气候特征 |
| --- | --- |
| 北海道 | 亚寒带，冬季严寒多雪 |
| 本州/四国/九州 | 温带季风，四季分明，梅雨季明显 |
| 冲绳/南西诸岛 | 亚热带，温暖湿润 |

- 四季分明，春樱、秋枫景致闻名。
- 6 月前后有「梅雨」季，夏秋多台风。

---

## 4. 自然资源

- 自然资源**相对贫乏**，能源与矿产高度依赖进口。
- 是世界主要的能源（石油、天然气、煤炭）进口国。
- 森林覆盖率高（约 68%），是世界森林覆盖率最高的发达国家之一。
- 周边海域渔业资源丰富，是渔业与水产消费大国。

## 5. 行政区划

日本实行**都道府县**制，共 **47 个一级行政区**：

| 类型 | 数量 | 说明 |
| --- | --- | --- |
| 都 | 1 | 东京都 |
| 道 | 1 | 北海道 |
| 府 | 2 | 大阪府、京都府 |
| 县 | 43 | 如神奈川县、爱知县等 |

- 下设市、町、村等基层自治体。

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 东京（Tokyo） | 首都，世界最大都市圈之一，政治经济中心 |
| 横滨（Yokohama） | 重要港口城市 |
| 大阪（Osaka） | 关西经济中心 |
| 名古屋（Nagoya） | 制造业（汽车）重镇 |
| 京都（Kyoto） | 千年古都，文化名城 |
| 札幌（Sapporo） | 北海道中心城市 |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 1.24 亿（2024 年） |
| 城市化率 | 约 92% |
| 人口趋势 | 持续下降，少子老龄化严重 |

- 人口高度集中于**太平洋沿岸的「太平洋工业带」**，尤其**东京都市圈**（约 3,700 万人，世界最大都市圈）。
- 山区与部分地方乡村人口稀少、持续流失。
- 是世界上**老龄化程度最高**的国家之一，65 岁以上人口占比近 30%。

---

## 📌 小结

日本是位于地震火山带上的狭长岛国，山多平原少，自然资源匮乏却拥有丰富的森林与海洋资源。人口与产业高度集中于太平洋沿岸的少数平原地带，东京都市圈是全球最大的城市群。频繁的自然灾害也深刻塑造了日本的防灾文化与社会韧性。
