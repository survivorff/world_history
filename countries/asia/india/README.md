# 🇮🇳 印度（Republic of India）

> 全称「印度共和国」，位于南亚，是世界上人口最多的国家、最大的民主制国家。拥有古老的文明、多元的宗教与文化，是增长最快的主要经济体之一，也是 IT 服务、航天与电影产业大国。

## 🧭 一句话画像

印度是拥有五千年文明的古国，也是当今世界人口最多的国家和规模最大的民主政体。它语言、宗教、民族极其多元，是印度教、佛教等宗教的发源地。近年来经济快速增长，在信息技术服务、制药、航天和电影（宝莱坞）等领域具有全球影响力，被广泛视为 21 世纪最具潜力的新兴大国之一。

## 🗺️ 速读要点

- **人口**：约 14.4 亿，2023 年超过中国成为世界第一人口大国，且人口结构年轻。
- **民主**：世界上人口规模最大的民主制国家，被称为「世界上最大的民主国家」。
- **经济**：世界第五大经济体，增长最快的主要经济体之一，IT 服务业全球领先。
- **多元**：语言、宗教、民族极度多样，印度教徒占多数。
- **历史**：古印度文明发源地，曾长期被英国殖民，1947 年独立。
- **军事**：核大国，军队规模世界前列。
