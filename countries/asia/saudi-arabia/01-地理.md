# 沙特阿拉伯 · 地理

## 1. 地理位置与疆域

沙特阿拉伯位于**西亚阿拉伯半岛**，占据半岛大部分面积。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 215 万平方公里（世界第 12，阿拉伯世界最大之一） |
| 陆地邻国 | 约旦、伊拉克、科威特、卡塔尔、阿联酋、阿曼、也门 |
| 濒临海域 | 红海（西）、波斯湾（东） |

- 是连接亚、非的战略要地。
- 扼守红海与波斯湾两大重要水道，地缘位置关键。

---

## 2. 地形地貌

沙特地形以**高原和沙漠**为主：

| 区域 | 特征 |
| --- | --- |
| 西部 | 沿红海的希贾兹—阿西尔山地，地势较高 |
| 中部 | 纳季德（Najd）高原 |
| 东部 | 沿波斯湾的低地平原（石油主产区） |
| 南部 | **鲁卜哈利沙漠（Rub' al Khali，「空旷的四分之一」）**，世界最大的连续沙漠之一 |

- 全境几乎没有常年河流和湖泊。
- 沙漠和半沙漠占据绝大部分国土。

---

## 3. 气候

沙特属典型的**热带沙漠气候**：

- 极度炎热干燥，夏季内陆气温常超 45℃。
- 降水稀少，多数地区年降水量极低。
- 昼夜温差大。
- 沿海地区湿度较高。
- 水资源极度匮乏，严重依赖**海水淡化**（世界最大的海水淡化国之一）。

---

## 4. 自然资源（石油王国）

- **石油**：拥有世界上最大的石油储量之一，是全球最大的石油出口国之一。东部省（尤其加瓦尔油田）是核心产区。
- **天然气**：储量丰富。
- 矿产：黄金、磷酸盐、铝土等。
- 资源高度集中于油气，是国家经济的绝对支柱。
- 大力发展太阳能（日照充足）。

---

## 5. 行政区划

沙特阿拉伯划分为 **13 个省（Region/Emirate）**：

- 各省由王室成员或任命的省长管理。
- 重要省份包括利雅得省、麦加省、东部省（石油）等。

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 利雅得（Riyadh） | 首都，最大城市，政治与商业中心 |
| 吉达（Jeddah） | 红海港口，通往麦加的门户 |
| 麦加（Mecca） | 伊斯兰教第一圣城，朝觐目的地 |
| 麦地那（Medina） | 伊斯兰教第二圣城 |
| 达曼（Dammam） | 东部石油工业中心 |
| 新未来城（NEOM） | 「2030 愿景」规划建设的未来城市 |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 3,700 万（2024 年） |
| 外籍人口 | 约占三分之一以上（大量外籍劳工） |
| 城市化率 | 约 84% |

- 人口集中于利雅得、麦加—吉达、东部省等城市地带。
- **外籍劳工**比例很高，来自南亚、东南亚、其他阿拉伯国家等，支撑经济运转。
- 本国人口结构年轻，年轻人占比高。

---

## 📌 小结

沙特阿拉伯雄踞阿拉伯半岛，是炎热干旱的沙漠之国，扼守红海与波斯湾两大要道。这里几乎没有河流，水资源依赖海水淡化，却埋藏着世界顶级的石油资源——这成为整个国家的命脉。麦加与麦地那两大圣城，则赋予其在伊斯兰世界无可替代的地位。
