## 1. 地理位置与疆域

法国本土位于**西欧**，是欧盟面积最大的国家，因轮廓近似六边形而被称为「**六边形（l'Hexagone）**」。

| 项目 | 数据 |
| --- | --- |
| 本土面积 | 约 55.2 万平方公里（欧盟第一） |
| 含海外领地 | 约 64 万平方公里 |
| 陆地邻国 | 比利时、卢森堡、德国、瑞士、意大利、摩纳哥、西班牙、安道尔 |
| 濒临海域 | 大西洋、地中海、英吉利海峡、北海 |

- 本土三面临海、一面靠山（阿尔卑斯、比利牛斯），地理条件优越。
- 拥有遍布全球的**海外领地**（见下文），使其成为唯一在多个大洋拥有领土的欧洲国家之一。

## 2. 地形地貌

法国本土地形多样：

| 区域 | 特征 |
| --- | --- |
| 北部与西部 | 平原与低地（巴黎盆地、卢瓦尔河谷） |
| 中部 | 中央高原（Massif Central），古老的火山高原 |
| 东部与东南 | 阿尔卑斯山脉、汝拉山 |
| 南部 | 比利牛斯山脉（与西班牙边界） |
| 地中海沿岸 | 蔚蓝海岸（普罗旺斯等） |

- 最高峰：**勃朗峰（Mont Blanc）**，海拔约 4,806 米，是西欧最高峰。
- 主要河流：塞纳河（流经巴黎）、卢瓦尔河（最长）、罗讷河、加龙河。

## 3. 气候

法国本土气候多样：

| 气候类型 | 分布 |
| --- | --- |
| 温带海洋性气候 | 西部与北部，温和湿润 |
| 半大陆性气候 | 东部与中部，冬冷夏热 |
| 地中海气候 | 南部沿海，夏干热、冬温和 |
| 高山气候 | 阿尔卑斯、比利牛斯山区 |

- 总体温和宜人，利于农业与葡萄种植。

---

## 4. 自然资源

- 矿产资源相对有限。
- 拥有丰富的**农业资源**：是欧盟最大的农业生产国，被誉为「欧洲粮仓」。
- **核能**：本土铀矿曾较丰富，核电是其能源支柱（占发电量约 70%，比例世界最高）。
- 水力、风能等可再生能源。
- 广阔的海洋经济专属区（得益于海外领地）。

---

## 5. 行政区划

法国是**单一制**国家，本土行政区划包括：

- **大区（Régions）**：本土 13 个 + 海外大区。
- **省（Départements）**：约 100 个。
- 下设市镇（Communes）等。

### 海外领地
- 法国拥有遍布全球的海外领地，包括**海外大区/省**（瓜德罗普、马提尼克、留尼汪、法属圭亚那、马约特）和**海外集体**（法属波利尼西亚、新喀里多尼亚等）。
- 这些领地使法国领土横跨大西洋、太平洋、印度洋。

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 巴黎（Paris） | 首都，最大城市，政治、经济、文化中心 |
| 马赛（Marseille） | 第二大城市，最大港口 |
| 里昂（Lyon） | 美食与工业中心 |
| 图卢兹（Toulouse） | 航空航天中心（空客总部） |
| 尼斯（Nice） | 蔚蓝海岸旅游城市 |
| 波尔多（Bordeaux） | 葡萄酒之都 |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 6,800 万（2024 年） |
| 城市化率 | 约 81% |
| 人口结构 | 生育率在欧洲较高，但仍趋于老龄化 |

- 人口分布相对均衡，巴黎大区（法兰西岛）人口最密集（约 1,200 万）。
- 是欧洲重要的移民接收国，拥有大量来自北非、西非及其他地区的移民及后裔。
- 生育率长期处于欧洲前列，人口结构相对年轻。

## 📌 小结

法国本土「六边形」三面环海、一面靠山，地形与气候多样，造就了「欧洲粮仓」般的农业优势和丰富的旅游资源。遍布三大洋的海外领地赋予其独特的全球地理存在和广阔的海洋专属经济区，是法国大国地位的重要组成部分。
