## 1. 地理位置与疆域

俄罗斯横跨**欧洲东部与亚洲北部**，是世界上面积最大的国家。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 1,710 万平方公里（世界第 1，约占全球陆地 1/8） |
| 时区 | 横跨 11 个时区 |
| 陆地边界 | 与 14 个国家接壤（含飞地加里宁格勒） |
| 海岸线 | 约 3.7 万公里 |
| 邻国 | 挪威、芬兰、爱沙尼亚、拉脱维亚、立陶宛、波兰、白俄罗斯、乌克兰、格鲁吉亚、阿塞拜疆、哈萨克斯坦、中国、蒙古、朝鲜 |
| 濒临海域 | 北冰洋、太平洋、波罗的海、黑海、里海等 |

- 地跨欧亚两洲，习惯上以**乌拉尔山脉**为欧亚分界，欧洲部分人口与经济较集中。
- **加里宁格勒州**是夹在波兰与立陶宛之间的飞地。

## 2. 地形地貌

俄罗斯地势总体**南高北低、东高西低**，主要地形单元：

| 地形 | 位置 | 特征 |
| --- | --- | --- |
| 东欧平原（俄罗斯平原） | 西部 | 人口与农业核心区 |
| 乌拉尔山脉 | 中部 | 欧亚分界，矿产丰富 |
| 西西伯利亚平原 | 中部 | 世界最大平原之一，多沼泽，富油气 |
| 中西伯利亚高原 | 中东部 | 高原与河谷 |
| 东西伯利亚与远东山地 | 东部 | 山地、火山（堪察加） |

- 最高峰：**厄尔布鲁士峰**（高加索，约 5,642 米），是欧洲最高峰。
- 主要河流：伏尔加河（欧洲最长河、母亲河）、鄂毕河、叶尼塞河、勒拿河、阿穆尔河（黑龙江）。
- **贝加尔湖**：世界最深、蓄水量最大的淡水湖。

## 3. 气候

由于纬度高、跨度大，俄罗斯以**寒冷**为主要气候特征：

| 气候带 | 分布 |
| --- | --- |
| 极地/苔原气候 | 北冰洋沿岸 |
| 亚寒带针叶林（泰加林）气候 | 西伯利亚大部 |
| 温带大陆性气候 | 欧洲部分与南部 |
| 局部温和气候 | 黑海沿岸（索契等地） |

- 大部分地区冬季漫长严寒，西伯利亚的奥伊米亚康是北半球「寒极」之一。
- 大面积**永久冻土**覆盖西伯利亚。

## 4. 自然资源

俄罗斯是**资源超级大国**：

- **天然气**：探明储量世界第一。
- **石油**：储量与产量均居世界前列。
- **矿产**：煤炭、铁、镍、钯、金、钻石、稀有金属等储量丰富。
- **森林**：森林面积世界第一，是「地球之肺」的重要组成。
- **水资源**：河流湖泊众多，贝加尔湖占全球未冻淡水的约 1/5。

能源出口是国家财政与外汇的核心支柱。

## 5. 行政区划

俄罗斯是**联邦制**国家，由多类「联邦主体」组成（共 80 余个，含存在国际争议的地区）：

| 类型 | 说明 |
| --- | --- |
| 共和国（Republics） | 多以主体民族命名，如鞑靼斯坦、车臣 |
| 边疆区（Krais） | 如滨海边疆区、克拉斯诺亚尔斯克边疆区 |
| 州（Oblasts） | 数量最多 |
| 联邦直辖市 | 莫斯科、圣彼得堡等 |
| 自治州 / 自治区 | 如犹太自治州 |

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 莫斯科（Moscow） | 首都，政治、经济、文化中心 |
| 圣彼得堡（Saint Petersburg） | 第二大城市，「北方威尼斯」，文化名城 |
| 新西伯利亚 | 西伯利亚最大城市 |
| 叶卡捷琳堡 | 乌拉尔工业重镇 |
| 符拉迪沃斯托克（海参崴） | 远东重要港口 |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 1.46 亿（2024 年） |
| 城市化率 | 约 75% |
| 人口趋势 | 总体增长乏力，部分年份负增长，老龄化 |

- 人口高度集中在**欧洲部分**（约占四分之三以上），尤其莫斯科与圣彼得堡都市圈。
- 西伯利亚与远东地广人稀。
- 面临人口下降、东部人口流失等结构性挑战。

---

## 📌 小结

俄罗斯以无与伦比的国土纵深和资源储备著称，欧洲部分是国家的人口、经济与政治心脏，广袤的西伯利亚则是能源与战略腹地。严寒气候与辽阔空间既塑造了独特的民族性格，也带来了开发与治理的巨大成本。
