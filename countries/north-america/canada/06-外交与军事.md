# 加拿大 · 外交与军事

[⬅ 返回加拿大目录](README.md) · [⬅ 返回主索引](../../../README.md)

---

## 1. 外交定位

加拿大外交以**多边主义**和**中等强国（Middle Power）**角色著称：

- 坚定支持以规则为基础的国际秩序和多边机构。
- 与美国保持**全球最紧密的双边关系之一**。
- 重视联合国维和、人道援助与人权外交。
- 是英联邦与法语国家组织的双重成员，具有独特的桥梁角色。

---

## 2. 主要双边与地区关系

| 对象 | 关系概述 |
| --- | --- |
| **美国** | 最重要的盟友与贸易伙伴，经济、安全、社会高度一体化 |
| **欧盟/英国** | 通过英联邦及自贸协定（CETA）保持紧密联系 |
| **法语国家** | 通过法语国家组织维系联系 |
| **中国** | 重要贸易伙伴，但关系近年因多重因素趋于复杂 |
| **印太地区** | 推出「印太战略」，加强参与 |

---

## 3. 国际组织与多边参与

- **联合国**：创始会员国，曾是维和理念（Peacekeeping）的重要倡导者与贡献国。
- **G7 / G20**：核心成员。
- **NATO（北约）**：创始成员。
- **英联邦（Commonwealth）**与**法语国家组织（Francophonie）**：双重成员。
- **USMCA**：北美自由贸易框架。
- **NORAD**：与美国共建的北美防空司令部。

---

## 4. 军事力量

加拿大武装力量（Canadian Armed Forces）规模适中、专业化。

| 项目 | 概况 |
| --- | --- |
| 规模 | 相对其国土较小，专业化程度高 |
| 军种 | 陆军、海军、空军 |
| 核武器 | **无核武器** |
| 特点 | 以北美防御、北约义务、国际维和与人道任务为主 |
| 北极 | 日益重视北极主权与安全 |

- 与美国通过 **NORAD（北美防空防天司令部）**共同负责北美空天防御。
- 参与北约在欧洲的部署及多项国际行动。
- 北极地区的主权与安全是日益重要的战略关注点。

---

## 5. 安全与外交焦点

- 维护与美国的特殊关系，同时管理贸易摩擦。
- **北极主权**与气候变化带来的航道、资源问题。
- 参与北约对乌克兰的支持。
- 推进「印太战略」，实现伙伴多元化。

---

## 6. 软实力与对外影响

- 多元文化、包容开放的国家形象。
- 维和与人道主义传统。
- 高质量的高等教育吸引国际学生。
- 文化（音乐、影视）与冰球的影响。

---

## 📌 外交与军事小结

加拿大以「中等强国」自居，是多边主义和联合国维和的坚定支持者。它与美国保持着世界上最紧密的双边与防务关系（NORAD），同时通过英联邦、法语国家组织发挥独特的桥梁作用。其军队规模适中、以防御和国际合作为主，而北极主权正成为其日益重要的战略议题。
