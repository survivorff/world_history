# 🇹🇷 土耳其（Republic of Türkiye）

> 全称「土耳其共和国」，地跨欧亚两洲，扼守黑海与地中海之间的战略要冲。是奥斯曼帝国的继承者、北约重要成员、新兴经济体和地区大国，以横跨东西的独特文明、伊斯坦布尔的历史地位和活跃的地缘外交著称。

## 🧭 一句话画像

土耳其是横跨欧亚的「文明十字路口」，前身是统治三大洲数百年的奥斯曼帝国。1923 年凯末尔建立世俗共和国，推行西化改革。它扼守博斯普鲁斯海峡，是北约重要成员和地区大国，在中东、巴尔干、高加索事务中扮演活跃角色。近年来在埃尔多安领导下，政治上转向总统制和更强的伊斯兰色彩，外交上奉行自主多元路线。

## 🗺️ 速读要点

- **地跨欧亚**：扼守博斯普鲁斯与达达尼尔海峡，连接黑海与地中海，战略地位极其重要。
- **历史**：奥斯曼帝国的继承者，1923 年凯末尔建立世俗共和国。
- **政治**：2017 年公投后转为总统制，埃尔多安长期主政。
- **军事**：北约第二大军队（按规模），国防工业（无人机等）崛起。
- **经济**：新兴经济体，制造业较强，但近年深受高通胀与里拉贬值困扰。
- **文化**：东西方文明交汇，伊斯坦布尔横跨两大洲。
