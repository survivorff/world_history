## 1. 地理位置与疆域

加拿大位于**北美洲北部**，是世界第二大国土面积的国家，北抵北冰洋。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 998 万平方公里（世界第 2） |
| 陆地邻国 | 仅美国（南部边界 + 西北与阿拉斯加） |
| 濒临海域 | 北冰洋、大西洋、太平洋 |
| 海岸线 | 约 20.2 万公里（**世界最长**） |
| 美加边界 | 约 8,890 公里，世界最长的不设防陆地边界 |

- 三面环洋，国土从大西洋延伸到太平洋，并深入北极地区。
- 是唯一同时濒临三大洋的美洲国家之一。

---

## 2. 地形地貌

| 区域 | 特征 |
| --- | --- |
| 西部 | 落基山脉（Rocky Mountains）等高大山系 |
| 中部 | 广阔的大草原（Prairies），农业带 |
| 东部 | 加拿大地盾、五大湖—圣劳伦斯低地 |
| 北部 | 北极群岛、苔原 |

- 最高峰：洛根峰（Mount Logan，约 5,959 米），位于育空地区。
- 拥有**全球最多的湖泊**，淡水资源占世界相当大比例。
- 主要河流：圣劳伦斯河、马更些河。

---

## 3. 气候

加拿大纬度高，气候以**寒冷**为主要特征：

| 区域 | 气候 |
| --- | --- |
| 北部 | 极地/苔原气候，严寒漫长 |
| 中部草原 | 大陆性气候，温差极大 |
| 东部 | 湿润大陆性气候，四季分明 |
| 西海岸（温哥华） | 温带海洋性气候，较温和湿润 |

- 大部分地区冬季漫长严寒，是世界上最寒冷的国家之一。
- 北部大面积为永久冻土。

---

## 4. 自然资源

加拿大是**资源超级大国**：

- **能源**：石油储量世界前列（阿尔伯塔油砂）、天然气、水电资源丰富。
- **矿产**：钾盐（世界最大）、铀、镍、黄金、钻石、铁等。
- **森林**：森林覆盖广阔，是世界主要木材与纸浆生产国。
- **淡水**：拥有世界上最丰富的淡水资源之一。
- **渔业**：漫长的海岸线带来丰富渔业资源。

## 5. 行政区划

加拿大是**联邦制**国家，由 **10 个省 + 3 个地区**组成：

| 类型 | 名称 |
| --- | --- |
| 10 个省 | 安大略、魁北克、不列颠哥伦比亚、阿尔伯塔、马尼托巴、萨斯喀彻温、新斯科舍、新不伦瑞克、纽芬兰与拉布拉多、爱德华王子岛 |
| 3 个地区 | 育空、西北地区、努纳武特 |

- 省拥有较大的自治权；地区的权力由联邦授予。
- **魁北克省**以法语为主，具有独特的文化地位。

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 渥太华（Ottawa） | 首都 |
| 多伦多（Toronto） | 最大城市，金融与商业中心 |
| 蒙特利尔（Montreal） | 魁北克最大城市，法语文化中心 |
| 温哥华（Vancouver） | 西海岸门户，太平洋枢纽 |
| 卡尔加里（Calgary） | 能源产业中心 |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 4,100 万（2024 年） |
| 人口密度 | 极低（约 4 人/平方公里） |
| 城市化率 | 约 82% |

- 约 **90% 的人口居住在距美国边境约 160 公里以内**的南部地带。
- 北部广袤地区人烟稀少。
- 人口增长主要依靠**移民**，是世界上移民比例最高的发达国家之一。

---

## 📌 小结

加拿大坐拥世界第二大的辽阔国土和世界最长的海岸线，能源、矿产、森林和淡水资源极为丰富。然而严寒的气候使其人口稀少且高度集中在南部美加边境附近。这种「地大物博、人聚南陲」的格局，深刻塑造了加拿大的经济结构与对美关系。
