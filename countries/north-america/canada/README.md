# 🇨🇦 加拿大（Canada）

> 位于北美洲北部，是世界第二大国土面积国家。资源极其丰富，社会高度发达、多元包容，是 G7 成员和世界主要发达经济体之一，以辽阔的自然风光、双语文化和高福利、高移民的社会模式著称。

## 🧭 一句话画像

加拿大是世界第二大国土面积的国家，自然资源（能源、矿产、森林、淡水）极为丰富，人口却相对稀少且高度集中在美加边境附近。作为 G7 成员和高度发达的福利国家，它以英法双语、多元文化和开放的移民政策著称，社会安定、生活质量位居世界前列，与美国保持着全球最紧密的经贸与安全关系之一。

## 🗺️ 速读要点

- **国土**：世界第二大国，海岸线世界最长，森林与淡水资源极丰富。
- **人口**：约 4,100 万，约 90% 居住在距美国边境约 160 公里内。
- **经济**：发达的资源型 + 服务型经济，与美国深度一体化。
- **政治**：君主立宪下的联邦议会制，魁北克的法语特性与自治诉求是长期议题。
- **社会**：英法双语、官方多元文化政策、高移民、高福利。
- **国际**：G7 成员，奉行多边主义与维和传统。
