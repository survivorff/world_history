# 🇸🇦 沙特阿拉伯（Kingdom of Saudi Arabia）

> 全称「沙特阿拉伯王国」，位于阿拉伯半岛，是伊斯兰教的诞生地和两大圣城（麦加、麦地那）所在地。坐拥世界顶级石油储量，是全球最大石油出口国之一、G20 成员和阿拉伯世界的重要大国，正通过「2030 愿景」推动经济多元化转型。

## 🧭 一句话画像

沙特阿拉伯是阿拉伯半岛的主导国家，伊斯兰教的发源地，拥有麦加和麦地那两大圣城，对全球穆斯林具有独特的宗教地位。它坐拥世界顶级的石油储量，是 OPEC 的核心和「石油市场的中央银行」。近年来，在王储穆罕默德·本·萨勒曼主导下，沙特通过「2030 愿景」大力推进社会改革和经济多元化，力图摆脱对石油的依赖。

## 🗺️ 速读要点

- **宗教**：伊斯兰教发源地，麦加、麦地那两大圣城所在，每年迎接全球朝觐者。
- **石油**：世界顶级石油储量与出口，沙特阿美是全球最赚钱的公司之一。
- **政治**：绝对君主制，沙特王室（沙特家族）统治，以伊斯兰教法为基础。
- **改革**：「2030 愿景」推动经济多元化与社会开放（如允许女性驾车、发展旅游与娱乐）。
- **地区**：阿拉伯世界与海湾地区的领导力量，OPEC 核心。
- **挑战**：经济转型、青年就业、地区安全与地缘竞争。
