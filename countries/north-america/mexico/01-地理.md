# 墨西哥 · 地理

## 1. 地理位置与疆域

墨西哥位于**北美洲南部**，是连接北美与中美洲的桥梁。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 196 万平方公里（世界第 13） |
| 陆地邻国 | 美国（北）、危地马拉、伯利兹（南） |
| 濒临海域 | 太平洋（西）、墨西哥湾与加勒比海（东） |
| 美墨边界 | 约 3,145 公里 |

- 北部与美国接壤，是世界上跨境往来最频繁的边界之一。
- 拥有漫长的太平洋与大西洋（墨西哥湾）海岸线。

---

## 2. 地形地貌

墨西哥地形以**高原和山脉**为主：

| 区域 | 特征 |
| --- | --- |
| 中部 | 墨西哥高原（核心人口与经济区） |
| 两侧 | 东马德雷山脉、西马德雷山脉 |
| 南部 | 火山带（横贯火山轴） |
| 沿海 | 沿太平洋与墨西哥湾的平原 |
| 半岛 | 北部下加利福尼亚半岛、东南尤卡坦半岛 |

- 多火山与地震（位于环太平洋地震带），如波波卡特佩特火山。
- 最高峰：奥里萨巴峰（Pico de Orizaba，约 5,636 米）。

---

## 3. 气候

墨西哥气候多样，南北、高低差异大：

| 区域 | 气候 |
| --- | --- |
| 北部 | 干旱、半干旱（沙漠） |
| 中部高原 | 温和（高海拔调节） |
| 南部与沿海低地 | 热带气候 |
| 尤卡坦/加勒比沿岸 | 热带，易受飓风影响 |

- 大致分旱季和雨季。
- 沿海地区受飓风影响。

---

## 4. 自然资源

- **石油与天然气**：墨西哥湾油田丰富，是重要产油国（国家石油公司 Pemex）。
- **矿产**：**白银**（世界最大产银国之一）、铜、金、锌、铅。
- **农业**：玉米（起源地）、牛油果（世界最大出口国）、番茄、辣椒、咖啡、龙舌兰（用于制龙舌兰酒）。
- 生物多样性极为丰富（「巨大多样性国家」之一）。

---

## 5. 行政区划

墨西哥是**联邦制**国家，由 **32 个联邦实体**组成（31 个州 + 墨西哥城）：

- 各州拥有自己的宪法、州长与议会。
- 墨西哥城是联邦首都，具有特殊地位。

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 墨西哥城（Mexico City） | 首都，最大城市，政治经济文化中心 |
| 瓜达拉哈拉（Guadalajara） | 第二大都市区，科技产业 |
| 蒙特雷（Monterrey） | 北部工业与商业重镇 |
| 蒂华纳（Tijuana） | 美墨边境制造业城市 |
| 坎昆（Cancún） | 加勒比海著名旅游城市 |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 1.29 亿（2024 年，世界第 10） |
| 城市化率 | 约 81% |
| 特点 | 西班牙语世界人口最多的国家 |

- 人口高度集中于**中部高原**（墨西哥城大都市区超 2,000 万人）。
- 北部边境城市因制造业和对美往来人口增长。
- 大量墨西哥裔移民居住在美国，跨境联系紧密。

---

## 📌 小结

墨西哥地处北美与中美洲的连接处，地形以高原山脉为主，火山地震频发，气候从北部沙漠到南部热带跨度极大。它是玉米的起源地、世界最大产银国之一，石油与生物多样性资源丰富。人口集中于中部高原，而与美国漫长的边界则深刻塑造了其经济与社会。
