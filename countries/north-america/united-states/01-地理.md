## 1. 地理位置与疆域

美国主体位于北美洲中部，由三大地理板块构成：

- **本土 48 州（Contiguous United States）**：位于北美大陆中部，东临大西洋，西濒太平洋，北接加拿大，南邻墨西哥及墨西哥湾。
- **阿拉斯加州（Alaska）**：位于北美大陆西北端，与加拿大接壤，隔白令海峡与俄罗斯相望，是面积最大的州。
- **夏威夷州（Hawaii）**：位于太平洋中部的火山群岛，是唯一完全位于热带、由岛屿组成的州。

此外还拥有若干**海外领地**：波多黎各、关岛、美属维尔京群岛、北马里亚纳群岛、美属萨摩亚等。

| 项目 | 数据 |

| 陆地边界 | 与加拿大边界约 8,890 公里（世界最长不设防陆地边界之一），与墨西哥边界约 3,145 公里 |

## 2. 地形地貌

美国地形自东向西大致可分为几大区域：

1. **大西洋沿岸平原**：东部沿海低平地带，向南延伸至墨西哥湾沿岸。
2. **阿巴拉契亚山脉（Appalachian Mountains）**：东部古老山系，较为低缓，曾是早期西进的天然屏障。
3. **中央大平原（Great Plains）与内陆低地**：广阔的平原和密西西比河流域，是世界级农业带。
4. **落基山脉（Rocky Mountains）**：西部高大山系，构成北美大陆分水岭。
5. **山间高原与盆地**：科罗拉多高原、大盆地等，地形干旱，著名的大峡谷位于此区。
6. **太平洋沿岸山系**：内华达山脉、海岸山脉，及肥沃的加州中央谷地。

### 主要河流与湖泊

- **密西西比河（Mississippi River）**：北美最长河流系统之一，与密苏里河合计长度位居世界前列，是核心航运与农业动脉。
- **密苏里河、俄亥俄河、科罗拉多河、里奥格兰德河**等。
- **五大湖（Great Lakes）**：苏必利尔湖、密歇根湖、休伦湖、伊利湖、安大略湖，与加拿大共享，是世界最大的淡水湖群。

## 3. 气候

美国幅员辽阔，几乎涵盖所有气候类型：

| 区域 | 气候特征 |
| --- | --- |
| 东南部 | 湿润亚热带，夏季炎热潮湿，易受飓风影响 |
| 东北部与五大湖区 | 湿润大陆性气候，四季分明，冬季寒冷 |
| 大平原 | 半干旱，温差大，是龙卷风高发的「龙卷风走廊」 |
| 西南部 | 干旱与半干旱，沙漠气候（如亚利桑那、内华达） |
| 西海岸（加州） | 地中海式气候，温和干燥 |
| 太平洋西北 | 海洋性气候，多雨湿润 |
| 阿拉斯加 | 亚寒带与极地气候 |
| 夏威夷 | 热带气候 |

**自然灾害**：东南沿海与墨西哥湾的飓风、中部的龙卷风、西部的地震与山火、密西西比流域的洪水等。

## 4. 自然资源

美国自然资源极为丰富：

- **能源**：页岩油气革命使其成为世界最大石油与天然气生产国之一；煤炭储量丰富。
- **矿产**：煤、铜、铅、钼、磷酸盐、铀、金、铁等。
- **耕地与水资源**：中部大平原是世界粮仓，密西西比水系提供灌溉与航运。
- **森林**：覆盖约三分之一国土，木材资源丰富。

## 5. 行政区划

- **50 个州（States）**：拥有独立的州宪法、州政府与立法机构。
- **1 个联邦特区**：华盛顿哥伦比亚特区（Washington, D.C.），首都所在，不属于任何州。
- **海外领地**：波多黎各、关岛等。
- 州以下设**县（County）**、**市（City）/镇（Township）**等地方政府。

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 3.35 亿（2024 年，世界第 3） |
| 城市化率 | 约 83% |
| 人口密度 | 较低（约 36 人/平方公里），分布极不均衡 |

- 人口高度集中于**东北走廊**（波士顿—纽约—费城—华盛顿）、**五大湖区**、**加州沿海**、**得克萨斯**与**佛罗里达**等地。
- 内陆山区与沙漠地带人口稀疏。
- 近几十年人口持续向**南部与西部「阳光地带」（Sun Belt）**迁移。

## 📌 小结

美国凭借辽阔且多样的地理空间、丰富的自然资源、便利的水陆通道，奠定了其农业、工业与战略安全的天然优势。东西两洋的地缘屏障也使其本土在历史上较少遭受外部直接军事威胁。
