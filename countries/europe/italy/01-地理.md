## 1. 地理位置与疆域

意大利位于**南欧**，主体是伸入地中海的**亚平宁半岛**，因形似靴子而被称为「靴子国」。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 30.2 万平方公里 |
| 陆地邻国 | 法国、瑞士、奥地利、斯洛文尼亚 |
| 国中之国 | 梵蒂冈、圣马力诺（被意大利领土包围） |
| 濒临海域 | 地中海（第勒尼安海、亚得里亚海、爱奥尼亚海等） |
| 主要岛屿 | 西西里岛、撒丁岛（地中海两大岛） |

- 北部以阿尔卑斯山与邻国相隔。
- 半岛与岛屿构成漫长的海岸线。

## 2. 地形地貌

意大利以**多山**著称：

| 区域 | 特征 |
| --- | --- |
| 北部 | 阿尔卑斯山脉与波河平原（最重要的农业与工业区） |
| 半岛 | 亚平宁山脉（Apennines）纵贯，构成「脊梁」 |
| 沿海与岛屿 | 平原狭小，多丘陵 |

- 最高点：勃朗峰意大利一侧（与法国接壤，超过 4,800 米）。
- **火山活跃**：维苏威火山（庞贝古城毁于其喷发）、埃特纳火山（欧洲最高活火山，西西里）、斯特龙博利火山。
- 主要河流：波河（Po，最长）、台伯河（流经罗马）。
- 位于地震带，地震时有发生。

## 3. 气候

意大利大部分属**地中海气候**：

| 区域 | 气候 |
| --- | --- |
| 沿海与南部 | 地中海气候，夏季炎热干燥，冬季温和湿润 |
| 北部波河平原 | 偏大陆性，冬冷夏热 |
| 阿尔卑斯山区 | 高山气候 |

- 总体温暖宜人，利于葡萄、橄榄、柑橘种植，也带来丰富的旅游资源。

---

## 4. 自然资源

- 矿产与能源资源**相对匮乏**，能源较依赖进口。
- 有少量天然气、大理石（卡拉拉白色大理石世界闻名）。
- 农业资源：适宜种植葡萄、橄榄、番茄、柑橘等地中海作物。
- 漫长海岸与文化遗产带来巨大的旅游资源。

## 5. 行政区划

意大利是**单一制**国家，由 **20 个大区（Regioni）**组成：

- 其中 5 个为**特别自治区**（西西里、撒丁、特伦蒂诺-上阿迪杰、瓦莱达奥斯塔、弗留利-威尼斯朱利亚），享有更大自治权。
- 大区下设省与市镇（Comuni）。

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 罗马（Rome） | 首都，最大城市，「永恒之城」 |
| 米兰（Milan） | 经济与时尚、金融中心 |
| 那不勒斯（Naples） | 南部最大城市，披萨发源地 |
| 都灵（Turin） | 工业（汽车）中心 |
| 威尼斯（Venice） | 著名水城 |
| 佛罗伦萨（Florence） | 文艺复兴名城 |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 5,900 万（2024 年） |
| 城市化率 | 约 72% |
| 人口趋势 | 出生率极低，老龄化严重，人口下降 |

- 北部（伦巴第、波河平原一带）经济发达、人口密集。
- **南北差距**明显：北部富裕工业化，南部（Mezzogiorno）相对落后。
- 老龄化在世界上名列前茅，出生率长期低迷。

## 📌 小结

意大利「靴形」半岛深入地中海，北靠阿尔卑斯、脊梁是亚平宁山脉，多山而火山地震活跃。温暖的地中海气候孕育了葡萄、橄榄与灿烂的旅游资源。资源虽不丰富，但优越的区位、丰厚的文化遗产构成独特优势。北富南穷的「南北差距」与严重老龄化，是其长期的地理—社会课题。
