# 🇨🇳 中国（People's Republic of China）

> 全称「中华人民共和国」，位于东亚，是世界上人口第二多、陆地面积第三或第四大的国家，拥有约五千年连续文明史，当今世界第二大经济体与重要的政治、军事、科技大国。

## 🧭 一句话画像

中国是一个拥有悠久连续文明、庞大人口与广袤国土的东方大国。改革开放以来经济高速增长，已成为世界第二大经济体、第一大制造业国和第一大货物贸易国。它实行中国共产党领导下的社会主义制度，近年来在科技、基建、军事等领域快速发展，是塑造 21 世纪国际格局的关键力量。

## 🗺️ 速读要点

- **文明**：拥有约五千年有文字记载的连续文明史，是四大文明古国中唯一文明未中断者。
- **政治**：中国共产党领导的社会主义国家，实行人民代表大会制度。
- **经济**：世界第二大经济体，最大制造业国与货物贸易国，近年推动高质量发展与科技自立。
- **人口**：约 14 亿，2022 年起人口进入负增长，老龄化加快。
- **科技与基建**：高铁里程、5G、新能源、航天等领域世界领先。
- **军事**：拥有核武器，军费规模世界第二，海军规模快速扩张。
