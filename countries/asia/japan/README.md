# 🇯🇵 日本（Japan）

> 全称「日本国」，位于东亚的太平洋岛国，由数千个岛屿组成。是世界主要发达经济体之一、科技与制造业强国，以独特的传统文化与高度现代化并存著称，对全球流行文化（动漫、游戏等）影响深远。

## 🧭 一句话画像

日本是一个将悠久传统与尖端科技完美融合的国家。它在二战后从废墟中迅速崛起，创造了「经济奇迹」，成为世界主要发达经济体和制造业强国（汽车、电子、机械）。同时，它以独特的文化输出——动漫、游戏、料理、设计——在全球拥有庞大影响力，是名副其实的文化软实力大国。

## 🗺️ 速读要点

- **岛国**：由本州、北海道、九州、四国四大岛及数千小岛组成，多火山地震。
- **经济**：世界第四大经济体，以汽车、电子、机械、精密制造闻名。
- **政治**：君主立宪 + 议会内阁制，天皇为象征，自民党长期主导。
- **历史**：明治维新实现近代化，二战战败后在和平宪法下重建并崛起。
- **人口**：少子老龄化最严重的国家之一，人口持续下降。
- **文化**：动漫、游戏、电影、料理（和食列入世界遗产）享誉全球。
