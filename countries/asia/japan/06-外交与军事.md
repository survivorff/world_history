## 1. 外交定位

日本外交以**日美同盟**为基轴，特征包括：

- **日美同盟**是其安全与外交的核心支柱。
- 战后长期奉行「**和平国家**」路线与经济外交。
- 重视多边主义、自由贸易与「自由开放的印太」构想。
- 是重要的对外援助（ODA）国家。
- 与邻国（中、韩、俄）既有经贸合作，也存在历史与领土问题。

---

## 2. 主要双边与地区关系

| 对象 | 关系概述 |
| --- | --- |
| **美国** | 同盟基轴，安全依靠美国「核保护伞」，驻有大量美军 |
| **中国** | 最大贸易伙伴之一，经贸密切但存在历史、领土（钓鱼岛/尖阁诸岛）与安全分歧 |
| **韩国** | 邻国与重要伙伴，因历史问题关系时有波折，近年趋于改善并加强安全协作 |
| **俄罗斯** | 因「北方四岛/南千岛群岛」争议未签和平条约，俄乌冲突后关系恶化 |
| **东盟** | 重要经贸与投资伙伴 |
| **欧盟、印度、澳大利亚** | 价值观相近的合作伙伴 |

## 3. 国际组织与多边参与

- **联合国**：重要会员国与主要会费贡献国，长期争取成为安理会常任理事国。
- **G7 / G20**：核心成员，G7 中唯一的亚洲国家。
- **OECD、WTO、APEC**等。
- **CPTPP**：跨太平洋伙伴关系协定的主导推动者之一。
- **QUAD（四方安全对话）**：美日印澳协调机制成员。

---

## 4. 军事力量——自卫队

由于和平宪法第九条限制，日本不设传统意义上的「军队」，而是**自卫队（Self-Defense Forces, SDF）**。

| 项目 | 概况 |
| --- | --- |
| 性质 | 专守防卫，不拥有进攻性战略武器 |
| 军种 | 陆上自卫队、海上自卫队、航空自卫队 |
| 装备 | 现代化程度高，海上与空中力量尤其精良 |
| 核武器 | 不拥有，坚持「无核三原则」，依赖美国核保护伞 |
| 军费 | 规模居世界前列，近年大幅增加 |

### 安保政策调整
- 近年来日本调整安保政策：
  - 2015 年通过新安保法案，允许有限度行使「集体自卫权」。
  - 计划将防卫开支提升至 GDP 的 2%。
  - 探讨获得「反击能力」（对敌方导弹基地的打击能力）。
- 这些变化反映其应对地区安全环境变化的战略转向。

---

## 5. 安全与外交焦点

- 应对地区安全形势变化，强化日美同盟与多边安全合作。
- 钓鱼岛（尖阁诸岛）等领土与海洋权益问题。
- 朝鲜的导弹与核问题。
- 推动「自由开放的印太」，加强与美、澳、印及欧洲的协作。
- 历史问题对周边关系的持续影响。

---

## 6. 软实力与对外影响

- 强大的文化输出（动漫、游戏、料理、设计）。
- 高质量的对外援助与基础设施合作。
- 「日本制造」的技术与品质声誉。
- 在国际组织与全球治理中的积极角色。

---

## 📌 外交与军事小结

日本以日美同盟为基轴、以「和平国家」形象和经济外交为特色，是 G7 中唯一的亚洲成员。受和平宪法约束，其防卫力量以「专守防卫」的自卫队形式存在。面对变化的地区安全环境，日本正逐步调整安保政策、增加防卫投入，并通过 QUAD、CPTPP 等深化与印太及全球伙伴的合作。
