# 🇮🇹 意大利（Italian Republic）

> 全称「意大利共和国」，位于南欧地中海，是古罗马文明和文艺复兴的发源地。拥有世界上最多的联合国教科文组织世界遗产，是 G7 成员、世界主要发达经济体之一，以艺术、时尚、设计、美食和制造业闻名于世。

## 🧭 一句话画像

意大利是西方文明的重要摇篮——古罗马帝国和文艺复兴皆诞生于此。它拥有全球数量最多的世界遗产，是艺术、时尚、设计、汽车和美食的国度。作为 G7 成员和欧盟创始国，意大利是世界主要发达经济体，以中小企业和「意大利制造」的精湛工艺著称，同时也面临高债务、低增长和南北差距等结构性挑战。

## 🗺️ 速读要点

- **文明**：古罗马文明与文艺复兴的发源地，西方文化的核心源头之一。
- **世界遗产**：拥有全球数量最多的联合国教科文组织世界遗产。
- **经济**：G7 成员，制造业强国，以中小企业与「意大利制造」闻名。
- **政治**：议会制共和国，政府更迭频繁是其政治传统特征。
- **文化**：艺术、时尚（米兰）、设计、汽车（法拉利）、美食（披萨、意面）享誉全球。
- **挑战**：高政府债务、经济增长乏力、人口老龄化、南北经济差距。
