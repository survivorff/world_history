# 🇮🇩 印度尼西亚（Republic of Indonesia）

> 全称「印度尼西亚共和国」，是世界上最大的群岛国家（「千岛之国」），由约 1.7 万个岛屿组成。人口世界第四，是世界上穆斯林人口最多的国家、东南亚最大经济体、G20 成员和东盟的核心力量。

## 🧭 一句话画像

印度尼西亚是世界最大的群岛国家，横跨赤道，由上万个岛屿串联而成。它是世界第四人口大国、穆斯林人口最多的国家，也是东南亚最大经济体和东盟的领导力量。以「多元一体（Bhinneka Tunggal Ika）」为国训，印尼在数百个民族、语言和多元宗教的基础上维系统一，是 G20 中的新兴市场代表和重要的资源（镍、煤、棕榈油）大国。

## 🗺️ 速读要点

- **群岛**：世界最大群岛国家，约 1.7 万个岛屿，横跨三个时区。
- **人口**：世界第四人口大国，穆斯林人口全球最多。
- **经济**：东南亚最大经济体，G20 成员，资源（镍、煤、棕榈油）大国。
- **多元**：数百个民族与语言，以「多元一体」为国训。
- **新首都**：正从雅加达迁都至加里曼丹岛的努桑塔拉（Nusantara）。
- **地理**：位于「火环」上，火山地震频发。
