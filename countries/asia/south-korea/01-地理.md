## 1. 地理位置与疆域

韩国位于**东亚朝鲜半岛南部**，三面环海。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 10 万平方公里 |
| 陆地邻国 | 朝鲜（北部，以军事分界线相隔） |
| 隔海相望 | 中国（黄海以西）、日本（东海/朝鲜海峡以东） |
| 濒临海域 | 黄海（西）、东海/日本海（东）、南海（南） |

- 与朝鲜以**军事分界线（三八线附近）**和**非军事区（DMZ）**相隔。
- 半岛地理位置使其处于中、日、俄等大国之间的战略要冲。

---

## 2. 地形地貌

- 地形以**山地和丘陵**为主，约占国土的 70%。
- 山脉多集中在东部，地势东高西低。
- 平原主要分布在西部和南部沿海，是人口与农业集中区。
- 最高峰：汉拿山（济州岛，约 1,947 米）。
- 主要河流：汉江（流经首尔）、洛东江。
- 海岸线曲折，西南沿海多岛屿；**济州岛**是最大岛屿，著名的旅游胜地。

## 3. 气候

韩国属**温带季风气候**，四季分明：

| 季节 | 特征 |
| --- | --- |
| 春 | 温和，樱花盛开 |
| 夏 | 炎热多雨，有梅雨季和台风 |
| 秋 | 凉爽宜人，枫叶景致 |
| 冬 | 寒冷干燥，受西伯利亚冷空气影响 |

---

## 4. 自然资源

- 自然资源**相对匮乏**，能源（石油、天然气）和多数原材料高度依赖进口。
- 有少量钨、石墨、煤炭等矿产。
- 森林覆盖率较高（约 63%）。
- 资源贫乏使韩国走上「加工出口型」经济道路。

## 5. 行政区划

韩国行政区划包括：

- **特别市**：首尔特别市。
- **广域市**：釜山、仁川、大邱、光州、大田、蔚山。
- **特别自治市/道**：世宗特别自治市、济州特别自治道、江原特别自治道等。
- **道（省）**：京畿道、忠清道、全罗道、庆尚道等。

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 首尔（Seoul） | 首都，最大城市，政治经济文化中心 |
| 釜山（Busan） | 第二大城市，最大港口 |
| 仁川（Incheon） | 重要港口与国际机场所在地 |
| 大邱、大田、光州 | 区域中心城市 |
| 世宗 | 行政中心城市（部分政府部门所在） |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 5,160 万（2024 年） |
| 城市化率 | 约 81% |
| 人口趋势 | 进入负增长，少子老龄化严重 |

- 人口高度集中于**首尔都市圈（首尔—仁川—京畿道）**，该区域聚集了全国约一半人口。
- 是世界上人口密度较高的国家之一。
- **生育率全球最低之一**，人口萎缩与老龄化是严峻挑战。

---

## 📌 小结

韩国地处朝鲜半岛南部的战略要冲，山多平原少、资源匮乏，却凭借「加工出口」模式创造了经济奇迹。人口和产业高度集中在首尔都市圈。与朝鲜对峙的地缘格局以及全球最低的生育率，是韩国地理与社会层面最突出的特征。
