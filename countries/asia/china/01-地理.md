## 1. 地理位置与疆域

中国位于**亚洲东部、太平洋西岸**，是亚欧大陆东端的大国。

| 项目 | 数据 |
| --- | --- |
| 陆地面积 | 约 960 万平方公里（世界第 3／第 4） |
| 陆地边界 | 约 2.2 万公里，与 14 国接壤（世界最多之一） |
| 海岸线 | 大陆海岸线约 1.8 万公里 |
| 邻国 | 朝鲜、俄罗斯、蒙古、哈萨克斯坦、吉尔吉斯斯坦、塔吉克斯坦、阿富汗、巴基斯坦、印度、尼泊尔、不丹、缅甸、老挝、越南 |
| 濒临海域 | 渤海、黄海、东海、南海 |

- 最东端在黑龙江与乌苏里江主航道交汇处，最西端在帕米尔高原，东西时差约 5 个时区，但全国统一用北京时间。
- 南北跨纬度近 50 度，从寒温带到热带。

## 2. 地形地貌

中国地势**西高东低**，呈三级阶梯状分布：

| 阶梯 | 主要地形 | 平均海拔 |
| --- | --- | --- |
| 第一级阶梯 | 青藏高原（「世界屋脊」） | 4000 米以上 |
| 第二级阶梯 | 内蒙古高原、黄土高原、云贵高原及各大盆地（塔里木、准噶尔、四川等） | 1000–2000 米 |
| 第三级阶梯 | 东部平原与丘陵（东北平原、华北平原、长江中下游平原） | 500 米以下 |

地形类型多样：山地约占 33%，高原约 26%，盆地约 19%，平原约 12%，丘陵约 10%。

### 主要山脉与高峰
- **喜马拉雅山脉**：主峰**珠穆朗玛峰**海拔约 8,848.86 米，世界最高峰。
- 昆仑山、天山、秦岭、横断山、大兴安岭等。

### 主要河流与湖泊
- **长江**：亚洲第一长河，全长约 6,300 公里，世界第三长。
- **黄河**：中华民族「母亲河」，约 5,464 公里。
- 珠江、黑龙江、雅鲁藏布江等。
- 湖泊：青海湖（最大咸水湖）、鄱阳湖、洞庭湖、太湖等。

## 3. 气候

中国气候复杂多样，主要受**季风**影响：

| 气候类型 | 分布 |
| --- | --- |
| 温带季风气候 | 华北、东北 |
| 亚热带季风气候 | 秦岭—淮河以南的南方大部 |
| 热带季风气候 | 海南、雷州半岛、云南南部 |
| 温带大陆性气候 | 西北内陆 |
| 高原山地气候 | 青藏高原 |

特征：雨热同期，夏季高温多雨，冬季寒冷干燥；东南沿海夏秋多台风，西北干旱少雨。

## 4. 自然资源

- **矿产**：种类齐全，稀土储量与产量世界第一；煤炭、钨、锑、锡、钼等储量丰富。
- **能源**：煤炭为传统主力；水能资源世界第一（长江三峡为最大水电站）；近年来风电、光伏装机量世界第一。
- **生物**：物种丰富，是大熊猫、金丝猴、朱鹮等珍稀物种的家园。
- **耕地与水资源**：耕地总量大但人均偏少；水资源时空分布不均（南多北少），有南水北调工程调配。

## 5. 行政区划

中国实行**省、县、乡三级**为主的行政区划体系，省级行政区共 34 个：

| 类型 | 数量 | 举例 |
| --- | --- | --- |
| 省 | 23 | 广东、江苏、山东、四川、台湾等 |
| 自治区 | 5 | 内蒙古、新疆、西藏、广西、宁夏 |
| 直辖市 | 4 | 北京、上海、天津、重庆 |
| 特别行政区 | 2 | 香港、澳门（实行「一国两制」） |

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 北京 | 首都，政治、文化、国际交往中心 |
| 上海 | 最大城市，经济、金融、航运中心 |
| 深圳 | 改革开放窗口，科技创新中心 |
| 广州 | 华南门户，商贸中心 |
| 重庆 | 西部直辖市，工业重镇 |
| 香港 | 国际金融中心，特别行政区 |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 14.1 亿（2024 年，世界第 2） |
| 城镇化率 | 约 66%（2023 年） |
| 人口增长 | 2022 年起进入负增长 |

- 人口分布以「**胡焕庸线**」（黑河—腾冲线）为界，东南半壁人口稠密（占全国约 94%），西北半壁地广人稀。
- 人口高度集中于东部沿海、长三角、珠三角、京津冀、成渝等城市群。
- 面临**老龄化加剧**与**出生率下降**的人口结构挑战。

---

## 📌 小结

中国地域辽阔、地形多样、资源相对丰富，西高东低的三级阶梯地势塑造了大江东流的水系格局。季风气候带来雨热同期、利于农业，但水土资源分布不均与人口的东密西疏，是其发展中长期需要平衡的地理条件。
