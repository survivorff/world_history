## 1. 地理位置与疆域

印度位于**南亚**，是南亚次大陆的主体国家，三面环海。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 328.7 万平方公里（世界第 7） |
| 陆地邻国 | 巴基斯坦、中国、尼泊尔、不丹、孟加拉国、缅甸 |
| 海上邻国 | 斯里兰卡、马尔代夫 |
| 濒临海域 | 阿拉伯海、孟加拉湾、印度洋 |
| 海岸线 | 约 7,500 公里 |

- 北部以喜马拉雅山脉为天然屏障，南部为半岛伸入印度洋。
- 与巴基斯坦、中国存在领土争议（克什米尔等地区）。

---

## 2. 地形地貌

印度地形大致分为三大区域：

| 区域 | 特征 |
| --- | --- |
| **北部喜马拉雅山区** | 高大山脉，是众多河流发源地 |
| **中部印度河—恒河平原** | 由大河冲积而成的肥沃平原，人口最密集、农业核心区 |
| **南部德干高原（Deccan Plateau）** | 半岛主体，古老的高原与丘陵 |

- 主要河流：**恒河（Ganges）**（印度教圣河）、布拉马普特拉河、印度河（上游）、亚穆纳河、戈达瓦里河。
- 西部有塔尔沙漠，东西两侧有西高止山与东高止山。

## 3. 气候

印度大部分地区属**热带季风气候**，季风对农业至关重要：

| 季节 | 特征 |
| --- | --- |
| 凉季（冬季，12–2 月） | 凉爽干燥 |
| 热季（3–5 月） | 炎热干旱，气温可达 40℃ 以上 |
| 雨季（西南季风，6–9 月） | 降水集中，对农业生产决定性影响 |
| 季风退却（10–11 月） | 逐渐转凉 |

- 季风强弱直接影响农业收成与经济。
- 北部山区为高山气候，西北部干旱。

## 4. 自然资源

- **矿产**：煤炭（储量丰富）、铁矿石、铝土、锰、云母（产量世界前列）。
- **能源**：煤炭为主，油气部分依赖进口，大力发展太阳能。
- **耕地**：拥有世界上规模庞大的可耕地，农业人口众多。
- 生物多样性丰富（孟加拉虎、亚洲象等）。

## 5. 行政区划

印度是**联邦制**国家，由**邦（States）**与**中央直辖区（Union Territories）**组成：

- 共 28 个邦和 8 个中央直辖区。
- 各邦拥有自己的政府与立法机构。
- 中央直辖区由联邦政府直接或部分管理（如德里、昌迪加尔等）。

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 新德里（New Delhi） | 首都，政治中心 |
| 孟买（Mumbai） | 最大城市，金融与宝莱坞中心 |
| 班加罗尔（Bengaluru） | 「印度硅谷」，IT 中心 |
| 加尔各答（Kolkata） | 东部文化与商业中心 |
| 钦奈（Chennai） | 南部工业与汽车中心 |
| 海得拉巴（Hyderabad） | IT 与制药中心 |

## 6. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 14.4 亿（2024 年，**世界第 1**） |
| 城市化率 | 约 36%（仍以农村人口为主） |
| 人口结构 | 非常年轻，中位年龄约 28 岁，「人口红利」显著 |

- 2023 年超过中国成为世界第一人口大国。
- 人口高度集中于**恒河平原**（北方邦、比哈尔邦等）。
- 城镇化加速但仍较低，大量人口居于农村。
- 年轻的人口结构被视为重要的长期发展潜力。

---

## 📌 小结

印度地处南亚，北有喜马拉雅屏障，中部恒河平原沃野千里，南部德干高原构成半岛主体。热带季风气候使农业「看天吃饭」，季风强弱牵动全国经济。庞大而年轻的人口既是巨大的发展潜力（人口红利），也对就业、资源与城市化构成长期压力。
