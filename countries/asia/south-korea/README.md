# 🇰🇷 韩国（Republic of Korea）

> 全称「大韩民国」，位于东亚朝鲜半岛南部。从战后废墟迅速崛起为发达经济体，创造了「汉江奇迹」。是全球科技与制造业强国（半导体、电子、汽车、造船），并以「韩流」文化（K-pop、影视）风靡世界。

## 🧭 一句话画像

韩国是 20 世纪发展奇迹的典范——从朝鲜战争后的贫困国家，在几十年间跃升为高度发达的经济体（「汉江奇迹」）。它是半导体、智能手机、汽车、造船领域的全球领导者（三星、现代等），又凭借「韩流」——K-pop（如 BTS）、韩剧、电影（《寄生虫》）——成为世界级文化强国。同时面临朝鲜半岛安全、少子化与社会竞争压力等挑战。

## 🗺️ 速读要点

- **奇迹**：从战后贫困到发达经济体，「汉江奇迹」的典范。
- **科技**：半导体（三星、SK 海力士）、智能手机、显示屏、汽车、造船全球领先。
- **财阀**：三星、现代、LG、SK 等大型企业集团主导经济。
- **韩流**：K-pop、韩剧、韩国电影构成强大的全球文化软实力。
- **政治**：总统制共和国，1987 年实现民主化。
- **挑战**：世界最低生育率之一、激烈的社会竞争、朝鲜半岛安全。
