# 国家介绍模板

> 复制本目录结构以新增一个国家。将 `<国家名>` 替换为实际国家，删除本说明段落后填充内容。每个文件顶部保留导航链接。

---

## 目录结构

```
countries/<大洲>/<国家英文>/
├── README.md          # 速览页（核心数据卡片 + 导航）

```

## 各专题文件应覆盖的要点

### 01-地理
- 地理位置、邻国、海陆边界
- 地形地貌、主要山脉河流湖泊
- 气候类型与分区
- 自然资源
- 行政区划
- 人口分布与主要城市

### 02-历史
- 史前与古代
- 中世纪 / 王朝更替
- 近代转折（殖民、革命、独立、战争）
- 现代国家形成
- 当代重大事件（按时间线）

### 03-政治
- 政治体制与宪法
- 立法、行政、司法三权结构
- 主要政党与选举制度
- 中央与地方关系
- 当前政治格局

### 04-经济
- 经济总量与发展阶段
- 三大产业结构
- 支柱产业与代表性企业
- 货币、金融、贸易
- 区域发展与社会经济议题

### 05-文化与社会
- 民族构成与语言
- 宗教信仰
- 教育与科技
- 文学、艺术、影视、音乐
- 体育
- 饮食与风俗
- 节日
- 社会议题

### 06-外交与军事
- 外交政策与定位
- 主要双边关系
- 国际组织参与
- 军事力量概况
- 重要条约与安全格局
