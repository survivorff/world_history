# 🇲🇽 墨西哥（United Mexican States）

> 全称「墨西哥合众国」，位于北美洲南部，是拉丁美洲第二大经济体、世界人口第十大国。拥有玛雅、阿兹特克等古文明遗产和丰富的自然与文化资源，是 G20 成员、重要的制造业基地和美国的关键贸易伙伴。

## 🧭 一句话画像

墨西哥是连接北美与拉美的桥梁，拥有玛雅、阿兹特克等灿烂的古印第安文明遗产，融合了原住民与西班牙殖民的文化传统。作为拉美第二大经济体和 G20 成员，墨西哥凭借与美国接壤的区位和 USMCA 协定，成为重要的制造业基地（汽车、电子）和「近岸外包」热点。同时也面临毒品犯罪、贫富差距和移民等挑战。

## 🗺️ 速读要点

- **文明**：玛雅、阿兹特克等古印第安文明发源地，文化遗产丰富。
- **经济**：拉美第二大经济体，制造业（汽车、电子）出口基地。
- **区位**：与美国接壤，USMCA 成员，「近岸外包（nearshoring）」热点。
- **人口**：约 1.29 亿，世界第十，西班牙语世界人口最多的国家。
- **政治**：联邦总统制共和国，2024 年选出首位女总统。
- **挑战**：毒品犯罪与暴力、贫富差距、移民问题。
