# 🇬🇧 英国（United Kingdom）

> 全称「大不列颠及北爱尔兰联合王国」，由英格兰、苏格兰、威尔士和北爱尔兰四个构成国组成。曾是「日不落帝国」，是工业革命发源地、议会民主的摇篮，至今仍是世界主要经济、金融、文化与军事强国之一。

## 🧭 一句话画像

英国是现代议会民主制度和工业革命的发源地，曾建立人类历史上面积最大的殖民帝国，对全球的语言、法律、政治制度、体育和文化产生了深远影响。如今，它仍是联合国安理会常任理事国、核大国和世界级金融中心（伦敦），并在 2020 年正式脱离欧盟（Brexit）后探索新的国际定位。

## 🗺️ 速读要点

- **构成**：由英格兰、苏格兰、威尔士、北爱尔兰组成的联合王国，实行权力下放。
- **政治**：君主立宪 + 议会主权，无成文宪法，是「威斯敏斯特体系」的母国。
- **历史**：工业革命发源地，曾是「日不落帝国」，两次世界大战的主要战胜国。
- **经济**：以金融服务业为主导，伦敦是全球顶级金融中心。
- **脱欧**：2020 年正式脱离欧盟，重塑对外关系。
- **军事**：核大国，联合国安理会常任理事国，北约核心成员。
- **文化**：英语、莎士比亚、披头士、足球——具有全球性的文化影响力。
