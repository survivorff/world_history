## 1. 地理位置与疆域

印度尼西亚位于**东南亚**，地跨赤道，是世界上**最大的群岛国家**。

| 项目 | 数据 |
| --- | --- |
| 总面积 | 约 191 万平方公里（陆地，世界第 14） |
| 岛屿数量 | 约 1.7 万个（有人居住的约 6,000 个） |
| 陆地邻国 | 巴布亚新几内亚、东帝汶、马来西亚（同岛接壤） |
| 海上邻国 | 新加坡、菲律宾、澳大利亚等 |
| 濒临海域 | 太平洋、印度洋及众多内海 |

- 横跨赤道，地跨亚洲与大洋洲之间。
- 扼守**马六甲海峡**等重要国际航道，地缘位置极其重要。

## 2. 主要岛屿

| 岛屿 | 特点 |
| --- | --- |
| **爪哇岛（Java）** | 人口最稠密，首都雅加达所在，政治经济核心 |
| **苏门答腊岛（Sumatra）** | 第二大岛，资源丰富 |
| **加里曼丹岛（Borneo）** | 与马来西亚、文莱共享，新首都所在 |
| **苏拉威西岛（Sulawesi）** | 形状独特 |
| **新几内亚岛西部（巴布亚）** | 与巴新共享，原始森林广布 |
| **巴厘岛（Bali）** | 世界著名旅游胜地 |

---

## 3. 地形与火山

- 多山多火山，位于**环太平洋火山地震带（火环）**上。
- 拥有数十座活火山（如喀拉喀托火山、默拉皮火山），是世界上火山最多的国家之一。
- **地震、火山喷发、海啸**频发（如 2004 年印度洋大海啸）。
- 火山灰造就了爪哇等地肥沃的土壤。

---

## 4. 气候

- 大部分属**热带雨林气候**，全年高温多雨。
- 终年炎热，分为旱季和雨季（受季风影响）。
- 湿度高，植被茂密。

---

## 5. 自然资源

印尼资源极为丰富：

- **矿产**：**镍**（储量与产量世界第一，电池关键原料）、锡、铜、金、铝土。
- **能源**：煤炭（世界主要出口国）、天然气、石油。
- **农业**：**棕榈油**（世界最大生产国）、橡胶、咖啡、可可、香料。
- **森林与生物多样性**：拥有世界第三大热带雨林，生物多样性极为丰富（红毛猩猩、科莫多巨蜥等）。
- **渔业**：广阔海域，渔业资源丰富。

## 6. 行政区划

印尼是**单一制**国家，划分为约 **38 个省（Provinsi）**：

- 部分省份享有特别自治地位（如亚齐、日惹、巴布亚地区等）。
- 省下设县/市、区等。

### 主要城市

| 城市 | 地位 |
| --- | --- |
| 雅加达（Jakarta） | 现首都，最大城市，经济中心（爪哇岛） |
| 努桑塔拉（Nusantara） | 规划中的新首都（加里曼丹岛） |
| 泗水（Surabaya） | 第二大城市，重要港口 |
| 万隆（Bandung） | 爪哇重要城市 |
| 棉兰（Medan） | 苏门答腊最大城市 |
| 登巴萨（巴厘岛） | 旅游中心 |

> **迁都**：因雅加达人口过密、交通拥堵、地面沉降等问题，印尼正将首都迁往加里曼丹岛的努桑塔拉。

---

## 7. 人口分布

| 项目 | 数据 |
| --- | --- |
| 总人口 | 约 2.8 亿（2024 年，世界第 4） |
| 城市化率 | 约 58% |
| 人口结构 | 较年轻 |

- 人口分布极不均衡，**约一半以上人口集中在爪哇岛**（世界人口最稠密的大岛之一）。
- 其他岛屿相对人口稀疏。
- 政府长期推行向外岛迁移人口的政策。

---

## 📌 小结

印度尼西亚是横跨赤道的「千岛之国」，由上万个岛屿组成，扼守马六甲海峡等战略航道。地处「火环」使其火山地震频发，但也带来肥沃土壤。镍、煤、棕榈油等资源极其丰富。人口高度集中于爪哇岛，迁都努桑塔拉正是为缓解爪哇过载、推动区域均衡发展的重大举措。
